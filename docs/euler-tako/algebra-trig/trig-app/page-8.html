Vectors

A vector is a specific quantity drawn as a line segment with a arrow at one end pointing in a specific direction. It has an initial point where the vector starts and a terminal point where the vector ends.

A vector is defined by its magnitude, or length of the line, and its direction which is indicated by an arrowhead at the terminal point.

note

There are various symbols that can be used to represent a vector such as...

  • A lower case, boldfaced type (with or without an arrow on top) such as v\mathbf{v}v or v⃗\vec{v}v.
  • An upper case set of letters with an arrow on top such as AB→\overrightarrow{AB}AB where AAA is the initial point and BBB is the terminal point.
  • With angle brackets such as ⟨a,b⟩\langle a, b \rangle⟨a,b⟩ where aaa and bbb are the coordinates of the terminal point (a,b)(a, b)(a,b) and the initial point is at the origin (0,0)(0, 0)(0,0).

Standard Position​

For a vector to be in standard position, its initial point must be at the origin of the coordinate plane. This means when a vector is written using angle bracket such as ⟨a,b⟩\langle a, b \rangle⟨a,b⟩, it is in standard position because the initial point is at the origin (0,0)(0, 0)(0,0) and the terminal point is at (a,b)(a, b)(a,b).

We can change any vector into standard position by translating the vector so that its initial point is at the origin. Thus, if the initial point of a vector AB→\overrightarrow{AB}AB is A(x1,y1)A (x_1, y_1)A(x1​,y1​) and the terminal point is B(x2,y2)B (x_2, y_2)B(x2​,y2​), then the position vector can be found by calculating...

CD→=⟨x2−x1,y2−y1⟩\overrightarrow{CD} = \langle x_2 - x_1, y_2 - y_1 \rangleCD=⟨x2​−x1​,y2​−y1​⟩

where CCC is the initial point at the origin and DDD is the terminal point at (x2−x1,y2−y1)(x_2 - x_1, y_2 - y_1)(x2​−x1​,y2​−y1​).

note

Even though we change the position of a vector to be in standard position, the magnitude and direction of the vector remain unchanged meaning they are equivalent. This is because vectors are defined by their magnitude and direction, not by their specific location in the coordinate plane.

Magnitude and Direction​

A vector is defined by its magnitude and direction which means we need to have a way to calculate these two properties if we want to work with vector. This is relatively easy because we can find a vectors magnitude using the Pythagorean Theorem or the distance formula and we can find its direction using the inverse tangent function.

Given a position vector v=⟨a,b⟩\mathbf{v} = \langle a, b \ranglev=⟨a,b⟩, the magnitude is given by ∣v∣=a2+b2|\mathbf{v}| = \sqrt{a^2 + b^2}∣v∣=a2+b2​ and the direction is given by θ\thetaθ = tan⁡−1(ba)\tan^{-1} \left( \dfrac{b}{a} \right)tan−1(ab​) where θ\thetaθ is the angle between the vector and the positive xxx-axis.

Vector Addition​

The sum of two vectors u\mathbf{u}u and v\mathbf{v}v also known as vector addition produces a third vector u+v\mathbf{u + v}u+v which is the resultant vector. To find u+v\mathbf{u + v}u+v, we first draw the vector u\mathbf{u}u and from the terminal end of u\mathbf{u}u, we draw the vector v\mathbf{v}v. This means that the initial point of v\mathbf{v}v is at the terminal point of u\mathbf{u}u. Finally, we can draw the resultant vector u+v\mathbf{u + v}u+v from the initial point of u\mathbf{u}u to the terminal point of v\mathbf{v}v.

Vector subtraction is similar to vector addition. To find u−v\mathbf{u - v}u−v, we view it as u+(−v)\mathbf{u + (-v)}u+(−v) where −v-v−v is the vector v\mathbf{v}v with the opposite direction. This means that to find u−v\mathbf{u - v}u−v, we first draw the vector u\mathbf{u}u and from the terminal end of u\mathbf{u}u, we draw the vector −v-v−v. Finally, we can draw the resultant vector u−v\mathbf{u - v}u−v from the initial point of u\mathbf{u}u to the terminal point of −v-v−v.

Scalar Multiplication​

Scalar multiplication is the process of multiplying a vector by a scalar (a real number). To find the product, we multiply each component of the vector by the scalar. Thus, to multiply v=⟨a,b⟩\mathbf{v} = \langle a, b \ranglev=⟨a,b⟩ by a scalar kkk, we calculate...

kv=⟨ka,kb⟩k\mathbf{v} = \langle ka, kb \ranglekv=⟨ka,kb⟩

where kakaka and kbkbkb are the components of the resulting vector. The magnitude of the resulting vector is ∣k∣|k|∣k∣ times the magnitude of the original vector and its direction is the same as the original vector if kkk is positive and opposite if kkk is negative.

Example‾\underline{Example}Example​

We can find 3u3 \mathbf{u}3u by multiplying each component of u\mathbf{u}u by 333 which gives us...

3u=⟨3⋅5,3⋅4⟩=⟨15,12⟩\begin{array}{cccccccccccccc} 3 \mathbf{u} &=& \langle 3 \cdot 5, 3 \cdot 4 \rangle \\[1em] &=& \langle 15, 12 \rangle \end{array}3u​==​⟨3⋅5,3⋅4⟩⟨15,12⟩​

Therefore, 3u=⟨15,12⟩3 \mathbf{u} = \langle 15, 12 \rangle3u=⟨15,12⟩.

Component Form​

In some cases, it is helpful to break down a vector into its components. A vector can be expressed in component form as the sum of its horizontal and vertical components. The horizontal component is the projection of the vector onto the xxx-axis and the vertical component is the projection of the vector onto the yyy-axis.

For example, given a vector v=⟨3,4⟩\mathbf{v} = \langle 3, 4 \ranglev=⟨3,4⟩, its horizontal component is ⟨3,0⟩\langle 3, 0 \rangle⟨3,0⟩ and its vertical component is ⟨0,4⟩\langle 0, 4 \rangle⟨0,4⟩. Therefore, we can write v\mathbf{v}v in component form as v=⟨3,0⟩+⟨0,4⟩\mathbf{v} = \langle 3, 0 \rangle + \langle 0, 4 \ranglev=⟨3,0⟩+⟨0,4⟩ which is equivalent to v=⟨3,4⟩\mathbf{v} = \langle 3, 4 \ranglev=⟨3,4⟩.

Unit Vector​

In addition to finding a vector's components, it is often useful to find a vector's unit vector which is a vector with a magnitude of 111 that points in the same direction as the original vector. If v=⟨a,b⟩\mathbf{v} = \langle a, b \ranglev=⟨a,b⟩ is a nonzero vector, then its unit vector u\mathbf{u}u can be found by dividing each component of v\mathbf{v}v by the magnitude of v\mathbf{v}v which gives us...

u=v∣v∣=⟨a∣v∣,b∣v∣⟩\mathbf{u} = \dfrac{\mathbf{v}}{|\mathbf{v}|} = \left\langle \dfrac{a}{|\mathbf{v}|}, \dfrac{b}{|\mathbf{v}|} \right\rangleu=∣v∣v​=⟨∣v∣a​,∣v∣b​⟩

Unit vectors are also often defined in terms of components with a horizontal and vertical unit vector. The horizontal unit vector is written as i=⟨1,0⟩\mathbf{i} = \langle 1, 0 \ranglei=⟨1,0⟩ and is directed along the positive horizontal axis. While the vertical unit vector is written as j=⟨0,1⟩\mathbf{j} = \langle 0, 1 \ranglej=⟨0,1⟩ and is directed along the positive vertical axis.

The magnitude is always a scalar and so dividing by a scalar is equivalent to multiplying by the reciprocal of the scalar.
