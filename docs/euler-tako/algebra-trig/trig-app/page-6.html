Parametric Equations

Parametric Equations are equations that help define a curve using a new variable, called a parameter. Instead of expressing yyy directly in terms of xxx (or vice versa), both xxx and yyy are expressed in terms of a third variable, usually denoted as ttt. This method is particularly useful for simplifying complex shapes, modeling objects in motion and representing relationships where the variables depend on a hidden factor like time.

Parameterizing a Curve​

When an object moves along a curve in a given direction and in a given amount of time, we can describe its position given by xxx and yyy coordinates. However, both xxx and yyy vary over time making them functions of time. For this reason, we can add another variable ttt which is the parameter upon which both xxx and yyy depend on. So, when we are parameterizing a curve, we are translating a single equation in two variables, such as xxx and yyy, into an equivalent pair of equations in three variables, such as xxx, yyy, and ttt.

In mathematical terms, suppose ttt is a number on an interval III and the set of ordered pairs, (x(t),y(t))(x(t), y(t))(x(t),y(t)), where t∈It \in It∈I, x=f(t)x = f(t)x=f(t), and y=g(t)y = g(t)y=g(t) forms a curve on the parameter ttt. Then, x=f(t)x = f(t)x=f(t) and y=g(t)y = g(t)y=g(t) are called the parametric equations of the curve.

Example‾\underline{Example}Example​

Parameterize the curve given by x=y3−2yx = y^3 - 2yx=y3−2y.

There can be multiple ways to parameterize a curve as long as the resulting equations satisfy the original equation. We could always let x=tx = tx=t and solve for yyy in terms of ttt but this equation is already solved for xxx. This means if we let y=ty = ty=t, it would be much easier to express xxx in terms of ttt.

So, letting y=ty = ty=t, we can substitute this into the original equation to get x=t3−2tx = t^3 - 2tx=t3−2t.

Therefore, the parametric equations of the curve are x(t)=t3−2tx(t) = t^3 - 2tx(t)=t3−2t and y(t)=ty(t) = ty(t)=t.

note

Sometimes, when we are parameterizing a curve, the curve can not be a function because it fails the vertical line test. However, parametric equations can still represent these curves because they do not require yyy to be expressed as a function of xxx or vice versa. This means parametric equations allow us to describe more complex curves that would otherwise be impossible to represent with standard functions.
