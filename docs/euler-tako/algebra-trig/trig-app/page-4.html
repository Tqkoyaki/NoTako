Graphing Polar Equations

Just as a rectangular equation such as y=x2y = x^2y=x2 describes the relationship between xxx and yyy on a Cartesian coordinate plane, a polar equation describes the relationship between rrr and θ\thetaθ on a polar coordinate plane. In both of these systems, the equation is not the only way to represent the relationship because a graph can also be used. By graphing polar equations, we can visualize the relationship between rrr and θ\thetaθ and better understand the behavior of the equations.

Testing Symmetry​

An equation of a graph is symmetric with respect to an axis if we can fold the graph along that axis and have both halves match up perfectly. For polar graphs, we typically test for three types of symmetry: symmetry about the polar axis (the horizontal axis), symmetry about the line θ=π2\theta = \dfrac{\pi}{2}θ=2π​ (the vertical axis), and symmetry about the pole (the origin).

After substituting the appropriate values into the equation, if we end up with an equivalent equation, then the graph is symmetric about that axis. The tests are as follows...

  1. To test for symmetry about the polar axis, we replace (r,θ)\left(r, \theta\right)(r,θ) with (r,−θ)\left(r, -\theta\right)(r,−θ) in the equation.
  2. To test for symmetry about the line θ=π2\theta = \dfrac{\pi}{2}θ=2π​, we replace (r,θ)\left(r, \theta\right)(r,θ) with (−r,−θ)\left(-r, -\theta\right)(−r,−θ).
  3. To test for symmetry about the pole, we replace (r,θ)\left(r, \theta\right)(r,θ) with (−r,θ)\left(-r, \theta\right)(−r,θ).

Graphing Polar Equations​

To graph a polar equation, we can create a table of θ\thetaθ and rrr values and then plot the points on a polar coordinate plane. However, we can use some strategies like properties of symmetry and identifying key values of θ\thetaθ and rrr to make the process easier by requiring fewer points.

Identifying Zeros and Maxima​

To find the zeros of a polar equation, we can set r=0r = 0r=0 and solve for θ\thetaθ. On the other hand, to find the maxima, we substitute θ\thetaθ for a value that would result in the maximum value of the trigonometric function. For example, if the equation contains sin⁡θ\sin \thetasinθ, we would substitute θ=π2\theta = \dfrac{\pi}{2}θ=2π​ because sin⁡π2=1\sin \dfrac{\pi}{2} = 1sin2π​=1. On the contrary, if the equation contains cos⁡θ\cos \thetacosθ, we would substitute θ=0\theta = 0θ=0 because cos⁡0=1\cos 0 = 1cos0=1.

Polar Curves​

To be able to graph polar equations more easily, it helps to be familiar with the common types of curves that can be represented by polar equations. By recognizing the type of curve, we can use its properties to graph it more efficiently.

The most basic polar curve is the circle which are given by r=acos⁡θr = a \cos \thetar=acosθ or r=asin⁡θr = a \sin \thetar=asinθ where aaa is the diameter of the circle or the distance from the pole to the farthest point on the circumference.

note

For r=acos⁡θr = a \cos \thetar=acosθ, the center is (a2,0)\left(\dfrac{a}{2}, 0\right)(2a​,0) and for r=asin⁡θr = a \sin \thetar=asinθ, the center is (a2,π2)\left(\dfrac{a}{2}, \dfrac{\pi}{2}\right)(2a​,2π​).

Cardioids​

One of the classic polar curves is the cardioid which resembles a heart shape. The formulas that produce cardioids include r=a±bcos⁡θr = a \pm b \cos \thetar=a±bcosθ and r=a±bsin⁡θr = a \pm b \sin \thetar=a±bsinθ where a>0a > 0a>0, b>0b > 0b>0, and ab=1\dfrac{a}{b} = 1ba​=1.

Limacons​

Another classic polar curve is the limacon which is an old french word for "snail" which describes the shape of the curve.

One-loop limacons which are also refered to as dimpled limacons are given by the formulas r=a±bcos⁡θr = a \pm b \cos \thetar=a±bcosθ and r=a±bsin⁡θr = a \pm b \sin \thetar=a±bsinθ where a>0a > 0a>0, b>0b > 0b>0, and 1<ab<21 < \dfrac{a}{b} < 21<ba​<2.

On the other hand, inner-loop limacons are given by the formulas r=a±bcos⁡θr = a \pm b \cos \thetar=a±bcosθ and r=a±bsin⁡θr = a \pm b \sin \thetar=a±bsinθ where a>0a > 0a>0, b>0b > 0b>0, and a<ba < ba<b. The graph of the inner-loop limacon passes through the pole twice where it is once for the outer loop and once for the inner loop.

note

Cardioids are a member of the limacon family where ab=1\dfrac{a}{b} = 1ba​=1 and is the reason why we can see the similarities in the graphs.

Lemniscates​

The lemniscate is another type of the classic polar curves which resembles a figure-eight or an infinity symbol. The formulas that produce lemniscates include r2=a2cos⁡2θr^2 = a^2 \cos 2\thetar2=a2cos2θ and r2=a2sin⁡2θr^2 = a^2 \sin 2\thetar2=a2sin2θ where a≠0a \neq 0a=0.

note

The formula r2=a2sin⁡2θr^2 = a^2 \sin 2\thetar2=a2sin2θ produces a lemniscate that is symmetric with respect to the pole while the formula r2=a2cos⁡2θr^2 = a^2 \cos 2\thetar2=a2cos2θ produces a lemniscate that is symmetric with respect to the line θ=π2\theta = \dfrac{\pi}{2}θ=2π​ and the polar axis.

Rose Curves​

The next type of polar curve is the rose curve which produces a petal-like shape. This shape can be achieved by using the formulas r=acos⁡nθr = a \cos n\thetar=acosnθ and r=asin⁡nθr = a \sin n\thetar=asinnθ where a≠0a \neq 0a=0. If nnn is odd, then the rose curve will have nnn petals. However, if nnn is even, then the rose curve will have 2n2n2n petals.

Archimedes Spiral​

Finally, the Archimedes spiral is a polar curve given by r=θr = \thetar=θ where θ≥0\theta \geq 0θ≥0. For this curve, as θ\thetaθ increases, rrr also increases at a constant rate in an never ending spiral that never intersects itself.
