Linear Equations

A linear equation is an equation of a straight line that is written in one variable which means the degree of this equation is always 111.

Single Variable Equations​

To solve a linear equation, we need to find the solution set which consists of all the values that can be substituted into the variable which make the equation true. We can classify equations using the type of solution sets we may encounter...

  1. An identity equation is an equation which is true for any value substituted into the variable (ex. 5x=9x−4x(ex.\:5x = 9x - 4x(ex.5x=9x−4x which is equal to 5x=5x)5x = 5x)5x=5x).
  2. A conditional equation is an equation which is true for only some values of the variable. (ex. 2x+3=7(ex.\:2x + 3 = 7(ex.2x+3=7 which is equal to x=4)x = 4)x=4).
  3. An inconsistent equation is an equation which is true for no possible value of the variable. It has no solution. (ex. 2x+3=2x+5(ex.\:2x + 3 = 2x + 5(ex.2x+3=2x+5 which is equal to 3=53 = 53=5 and this is not a true statement))).
note

A linear equation in one variable can be always written in the form ax+b=0ax + b = 0ax+b=0 where aaa and bbb are real numbers, a≠0a \neq 0a=0. This means only conditional equations can be linear equations.

We can solve a linear equation by manipulating the equation in order to isolate the unknown operation. We are able to apply any operation to the equation as long as its applied to both sides in order to isolate the unknown variable.

Rational Equations​

Rational equations are equations that contain at least one rational expression and these equations after some manipulation can result in a linear equation.

One method to solving rational equations is to find the least common denominator (LCD) which is the smallest term which is divisible by all the denominators. By multiplying both sides by the LCD, we can cancel out all the denominators and then solve the resulting equation.

Finding Linear Equations​

The most used form of a linear equation is the slope-intercept form. This form is written as y=mx+by = mx + by=mx+b, where mmm is the slope and bbb is the y-intercept.

The slope of a line is the ratio of the vertical change in yyy over the horizontal change in xxx between any two points on a line. This change is often described by rise over run and indicates both the steepness of the line and its direction. If the slope is negative then yyy decreases as xxx increases. If the slope is positive then yyy increases as xxx increases.

Given any two points on the line, (x1,y1)(x_1, y_1)(x1​,y1​) and (x2,y2)(x_2, y_2)(x2​,y2​), we can find the slope using m=y2−y1x2−x1m = \frac{y_2 - y_1}{x_2 - x_1}m=x2​−x1​y2​−y1​​.

note

If we have any point and the slope of a line, we can find the y-intercept by substituting for xxx, yyy, and mmm in the slope-intercept form in order to solve for bbb.

Point-Slope Form​

If we have the slope and one point on a line, we can find the equation of a line using point-slope form. The form is y−y1=m(x−x1)y - y_1 = m(x - x_1)y−y1​=m(x−x1​) where the point is (x1,y1)(x_1, y_1)(x1​,y1​) and the slope is mmm.

Standard Form​

The equation of a line can also be written in standard form. This form is written as Ax+By=CAx + By = CAx+By=C where AAA, BBB, and CCC are integers meaning no fractions are allowed.

Horizontal & Vertical Lines​

The equation of a vertical line is x=cx = cx=c where ccc is a constant. For these lines, the slope is undefined and regardless of the yyy-value of any point, xxx will always equal ccc.

The equation of a horizontal line is y=cy = cy=c where ccc is a constant. For these lines, the slope is 000 and regardless of the xxx-value of any point, yyy will always equal ccc.

Parallel and Perpendicular​

Parallel lines are lines that have the same slope but different yyy-intercepts. This means that these lines will never intersect.

Perpendicular lines are lines that intersect to form a 90∘90^{\circ}90∘ angle. The slope of one line is the negative reciprocalof the other. In order words, if m1=am_1 = am1​=a then m2=−1am_2 = -\frac{1}{a}m2​=−a1​. We can show two lines are perpendicular if the product of the two slopes equal -1 as m1×m2=−1m_1 \times m_2 = -1m1​×m2​=−1.

We can find a parallel or perpendicular line passing through a given point by first finding the new slope and then using the slope and point to find the new equation.
