Graphing Sine and Cosine

We have already defined the trigonometric functions in terms of unit circles. Recall that these functions relate real number values to the xxx and yyy coordinates of a point on the unit circle. This means that we can also graph these functions on a coordinate plane which gives us another way to visualize and analyze these functions.

Periodic Functions​

Both the shapes of the sine and cosine graphs repeat after 2π2\pi2π units which means the functions are periodic with a period of 2π2\pi2π. A periodic function in general is a function for which a horizontal shift, PPP, results in f(x+P)=f(x)f(x + P) = f(x)f(x+P)=f(x) for all xxx in the domain of the function. The period of a periodic function is the smallest positive number PPP for which this is true. The period is the smallest such horizontal shift with P>0P > 0P>0.

Sinusoidal Functions​

A sinusoidal function is a function that has a general shape of a sine wave and both the sine and cosine functions are examples of sinusoidal functions. The general forms of sinusoidal functions are y=Asin⁡(Bx+C)+Dy = A\sin(Bx + C) + Dy=Asin(Bx+C)+D and y=Acos⁡(Bx+C)+Dy = A\cos(Bx + C) + Dy=Acos(Bx+C)+D where AAA, BBB, CCC, and DDD are constants.

note

The curves of sinusoidal functions are not necessarily identical as some can be taller or longer than others.

Period​

From the general forms of the sinusoidal functions, we can deduce that they are transformations of the sine and cosine functions and we can determine how each constant affects the graph of the function accordingly.

In the general formula, BBB is related to the period as P=2π∣B∣P = \dfrac{2\pi}{| B |}P=∣B∣2π​. If ∣B∣>1| B | > 1∣B∣>1, then the period is less than 2π2\pi2π and the graph is compressed horizontally. If ∣B∣<1| B | < 1∣B∣<1, then the period is greater than 2π2\pi2π and the graph is stretched horizontally. Finally, if B=1B = 1B=1, then the period is 2π2\pi2π.

Amplitude​

Other than horizontal compression and stretch, the graph can also be vertically compressed or stretched. In the general formula, AAA relates to the amplitude or the greatest distance from rest. The value of AAA directly represents the vertical stretch factor, and ∣A∣| A |∣A∣ is the amplitude of the function. This means that the local maxima will be ∣A∣| A |∣A∣ units above the horizontal midline of the graph and the local minima will be ∣A∣| A |∣A∣ units below the horizontal midline where the midline is the horizontal line that passes through the center of the graph and is given by y=Dy = Dy=D.

Also note that if ∣A∣>1| A | > 1∣A∣>1, then the graph is stretched vertically. If ∣A∣<1| A | < 1∣A∣<1, then the graph is compressed vertically. Finally, if A=1A = 1A=1, then the amplitude is 111.

note

Due to the amplitude being the vertical height from the midline, we can also find the amplitude using the formula ∣A∣=12∣max−min∣| A | = \dfrac{1}{2} |\text{max} - \text{min}|∣A∣=21​∣max−min∣ where the max\text{max}max and min\text{min}min are the maximum and minimum values of the function respectively.

Phase Shift​

Before we explore the phase shift, let's slightly modify the general forms of the sinusoidal functions from y=Asin⁡(Bx−C)+Dy = A\sin(Bx - C) + Dy=Asin(Bx−C)+D and y=Acos⁡(Bx−C)+Dy = A\cos(Bx - C) + Dy=Acos(Bx−C)+D to y=Asin⁡(B(x−CB))+Dy = A\sin(B(x - \dfrac{C}{B})) + Dy=Asin(B(x−BC​))+D and y=Acos⁡(B(x−CB))+Dy = A\cos(B(x - \dfrac{C}{B})) + Dy=Acos(B(x−BC​))+D. This is because the value of CB\dfrac{C}{B}BC​ represents the phase shift which is the horizontal shift of the graph. If C>0C > 0C>0, then the graph is shifted to the right by ∣C∣| C |∣C∣ units and if C<0C < 0C<0, then the graph is shifted to the left by ∣C∣| C |∣C∣ units.

Vertical Shift​

Finally, the value of DDD indicates the vertical shift of the graph which is the vertical translation of the graph. If D>0D > 0D>0, then the graph is shifted up by DDD units and if D<0D < 0D<0, then the graph is shifted down by ∣D∣| D |∣D∣ units.

Graphing Variations​

Given a sinusoidal function in the form y=Asin⁡(Bx−C)+Dy = A\sin(Bx - C) + Dy=Asin(Bx−C)+D or y=Acos⁡(Bx−C)+Dy = A\cos(Bx - C) + Dy=Acos(Bx−C)+D, we can find the period, amplitude, phase shift, and midline using the following formulas...

  1. The amplitude is ∣A∣| A |∣A∣.
  2. The period is 2π∣B∣\dfrac{2\pi}{| B |}∣B∣2π​.
  3. The phase shift is CB\dfrac{C}{B}BC​.
  4. The midline is y=Dy = Dy=D.

Given the function y=Asin⁡(Bx−C)+Dy = A\sin(Bx - C) + Dy=Asin(Bx−C)+D, we can sketch the graph by following these steps...

  1. Find the amplitude and period.
  2. We will graph y=sin⁡(Bx)y = \sin(Bx)y=sin(Bx) first and then will apply the phase shift and vertical shift.
  3. Start at the origin, with the function increasing to the right if AAA is positive and decreasing to the right if AAA is negative.
  4. At x=π2∣B∣x = \dfrac{\pi}{2 | B |}x=2∣B∣π​, the function will reach a local maximum if A>0A > 0A>0 and a local minimum if A<0A < 0A<0.
  5. The curve returns to the xxx-axis at x=π∣B∣x = \dfrac{\pi}{| B |}x=∣B∣π​.
  6. There is a local minimum for A>0A > 0A>0 and a local maximum for A<0A < 0A<0 at x=3π2∣B∣x = \dfrac{3\pi}{2 | B |}x=2∣B∣3π​ with y=−Ay = -Ay=−A.
  7. Finally, the curve returns to the xxx-axis at x=2π∣B∣x = \dfrac{2\pi}{| B |}x=∣B∣2π​.
  8. The rest of the graph can be sketched by following the same pattern.
  9. Now that we have the graph of y=sin⁡(Bx)y = \sin(Bx)y=sin(Bx), we can apply the phase shift of CB\dfrac{C}{B}BC​ units to the right if C>0C > 0C>0 and to the left if C<0C < 0C<0.
  10. Finally, we can apply the vertical shift of DDD units up if D>0D > 0D>0 and down if D<0D < 0D<0.

Cosine Form​

We can graph the cosine function in a similar way. Given the function y=Acos⁡(Bx−C)+Dy = A\cos(Bx - C) + Dy=Acos(Bx−C)+D, we can sketch the graph by following these steps...

  1. Find the amplitude and period.
  2. We will graph y=cos⁡(Bx)y = \cos(Bx)y=cos(Bx) first and then will apply the phase shift and vertical shift.
  3. Start at the point (0,A)(0, A)(0,A), with the function decreasing to the right if AAA is positive and increasing to the right if AAA is negative.
  4. At x=π2∣B∣x = \dfrac{\pi}{2 | B |}x=2∣B∣π​, the function will reach the xxx-axis.
  5. The function will reach a local minimum if A>0A > 0A>0 and a local maximum if A<0A < 0A<0 at x=π∣B∣x = \dfrac{\pi}{| B |}x=∣B∣π​.
  6. The function will return to the xxx-axis at x=3π2∣B∣x = \dfrac{3\pi}{2 | B |}x=2∣B∣3π​.
  7. Finally, there is a local maximum for A>0A > 0A>0 and a local minimum for A<0A < 0A<0 at x=2π∣B∣x = \dfrac{2\pi}{| B |}x=∣B∣2π​ with y=−Ay = -Ay=−A.
  8. The rest of the graph can be sketched by following the same pattern.
  9. Now that we have the graph of y=cos⁡(Bx)y = \cos(Bx)y=cos(Bx), we can apply the phase shift of CB\dfrac{C}{B}BC​ units to the right if C>0C > 0C>0 and to the left if C<0C < 0C<0.
  10. Finally, we can apply the vertical shift of DDD units up if D>0D > 0D>0 and down if D<0D < 0D<0.

Real World Applications​

We can use the transformations of sine and cosine functions in numerous applications especially anything related to circular motion.

Example‾\underline{Example}Example​

The London Eye is a huge Ferris wheel with a diameter of 135135135 meters (443443443 feet). It completes one rotation every 303030 minutes. Riders board from a platform 222 meters above the ground. Express a rider's height above the ground as a function of time in minutes.

With a diameter of 135135135 meters, the radius of the Ferris wheel is 67.567.567.5 meters. This means the height will osciallate between 67.567.567.5 meters above and below the center. This means the amplitude is 67.567.567.5 meters.

The passengers board 222 meters above the ground which means the center of the wheel is located 67.5+2=69.567.5 + 2 = 69.567.5+2=69.5 meters above the ground. This means the midline of the oscillation will be at 69.569.569.5 meters.

The wheel also takes 303030 minutes to complete one revolution which means the period is 303030 minutes.

Finally, since the rider boards at the lowest point, the height must start at the smallest possible value and start increasing from there. This shape is the shape of a vertically reflected cosine function.

The information we have extracted from the problem are that the amplitude is A=67.5A = 67.5A=67.5, the midline is D=69.5D = 69.5D=69.5, the period is P=30P = 30P=30, and the shape is −cos⁡(t)-\cos(t)−cos(t).

We can also find the value of BBB using the formula P=2π∣B∣P = \dfrac{2\pi}{| B |}P=∣B∣2π​ which gives us 30=2π∣B∣30 = \dfrac{2\pi}{| B |}30=∣B∣2π​. This means ∣B∣=2π30=π15| B | = \dfrac{2\pi}{30} = \dfrac{\pi}{15}∣B∣=302π​=15π​.

Putting all this information together, we get the function h(t)=−67.5cos⁡(π15t)+69.5h(t) = -67.5\cos(\dfrac{\pi}{15}t) + 69.5h(t)=−67.5cos(15π​t)+69.5 where ttt is in minutes and yyy is measured in meters.
