Inverse Trigonometric Functions

Through the use of trigonometric functions, we can find the ratio of sides of a right triangle given an angle. However, it is often useful and necessary to be able to find the angle given the ratio of sides. This is where inverse trigonometric functions come into play.

Inverse Sine, Cosine, and Tangent​

The inverse function "undoes" what the original function "does". However, only one-to-one functions can be inverted because we swap the domain and range between the two functions meaning the original function must pass the horizontal line test for the inverse to pass the vertical line test.

Inherently, the sine, cosine, and tangent functions are not one-to-one function as they would fail the horizontal line test. In fact, no periodic function can be one-to-one because each output in its range corresponds to at least one input in every period and there are infinitely many periods. To remedy this, we restrict the domain of each function to yield a new function that is one-to-one that can be inverted. These domain restrictions are somewhat arbitrary, but the conventional choices are as follows...

  • Sine: [−π2,π2]\text{Sine: } [-\dfrac{\pi}{2}, \dfrac{\pi}{2}]Sine: [−2π​,2π​]
  • Cosine: [0,π]\text{Cosine: } [0, \pi]Cosine: [0,π]
  • Tangent: (−π2,π2)\text{Tangent: } (-\dfrac{\pi}{2}, \dfrac{\pi}{2})Tangent: (−2π​,2π​)

Using these restrictions, we can define the inverse functions as follows:

  • The inverse sine function is defined as sin⁡−1(x)\sin^{-1}(x)sin−1(x) or arcsin⁡(x)\arcsin(x)arcsin(x) which means x=sin⁡(y)x = \sin(y)x=sin(y). This function is sometimes called the arc sine function. Also note that y=sin⁡−1(x)y = \sin^{-1}(x)y=sin−1(x) has domain [−1,1][-1, 1][−1,1] and range [−π2,π2][-\dfrac{\pi}{2}, \dfrac{\pi}{2}][−2π​,2π​].
  • The inverse cosine function is defined as cos⁡−1(x)\cos^{-1}(x)cos−1(x) or arccos⁡(x)\arccos(x)arccos(x) which means x=cos⁡(y)x = \cos(y)x=cos(y). This function is sometimes called the arc cosine function. Also note that y=cos⁡−1(x)y = \cos^{-1}(x)y=cos−1(x) has domain [−1,1][-1, 1][−1,1] and range [0,π][0, \pi][0,π].
  • The inverse tangent function is defined as tan⁡−1(x)\tan^{-1}(x)tan−1(x) or arctan⁡(x)\arctan(x)arctan(x) which means x=tan⁡(y)x = \tan(y)x=tan(y). This function is sometimes called the arc tangent function. Also note that y=tan⁡−1(x)y = \tan^{-1}(x)y=tan−1(x) has domain (−∞,∞)(-\infty, \infty)(−∞,∞) and range (−π2,π2)(-\dfrac{\pi}{2}, \dfrac{\pi}{2})(−2π​,2π​).
note

The inverse of any function can be found by reflecting the original function across the line y=xy = xy=x. This means that y=sin⁡−1(x)y = \sin^{-1}(x)y=sin−1(x) is the reflection of y=sin⁡(x)y = \sin(x)y=sin(x) across the line y=xy = xy=x. This is true for all inverse functions.

Also note that the notation sin⁡−1(x)\sin^{-1}(x)sin−1(x) is not the same as 1sin⁡(x)\dfrac{1}{\sin(x)}sin(x)1​. The latter is the cosecant function, which is defined as csc⁡(x)=1sin⁡(x)\csc(x) = \dfrac{1}{\sin(x)}csc(x)=sin(x)1​ or (sin⁡(x))−1(\sin(x))^{-1}(sin(x))−1. The notation sin⁡−1(x)\sin^{-1}(x)sin−1(x) is reserved for the inverse sine function.

Finding Exact Values​

Just as we did with the original trigonometric functions, we can give exact values for the inverse functions when we are using the special angles, specifically 0,π6,π4,π3,π20, \dfrac{\pi}{6}, \dfrac{\pi}{4}, \dfrac{\pi}{3}, \dfrac{\pi}{2}0,6π​,4π​,3π​,2π​, and their reflections into other quadrants.

Given a "special angle" input value, we can evalue an inverse trigonometric function by finding angle xxx such that the original trigonometric function has an output equal to the given input for the inverse trigonometric function. In the case that xxx is not defined in the range of the inverse function, we find another angle yyy such that the angle yyy is in the defined range of the inverse trigonometric function.

Example‾\underline{Example}Example​

Evalue cos⁡−1(12)\cos^{-1}(\dfrac{1}{2})cos−1(21​).

The cosine function is positive in the first and fourth quadrants but the range of the inverse cosine function is [0,π][0, \pi][0,π]. Therefore, we need to find an angle in the first quadrant that has a cosine value of 12\dfrac{1}{2}21​. This angle is π3\dfrac{\pi}{3}3π​.

The solution is cos⁡−1(12)=π3\cos^{-1}(\dfrac{1}{2}) = \dfrac{\pi}{3}cos−1(21​)=3π​.

Composite Functions​

As a refresher, a composite function is a function that is formed by combining two or more functions. The notation for a composite function is f(g(x))f(g(x))f(g(x)) which means that we first apply the function g(x)g(x)g(x) and then apply the function f(x)f(x)f(x) to the result of g(x)g(x)g(x). The order of the functions matters, so f(g(x))≠g(f(x))f(g(x)) \neq g(f(x))f(g(x))=g(f(x)) in general.

There are four cases to consider when dealing with trigonometric functions and their inverses. The first case is when the composition is of the form f(f−1(x))f(f^{-1}(x))f(f−1(x))...

  • sin⁡(sin⁡−1(x))=x  for  −1≤x≤1\sin(\sin^{-1}(x)) = x \:\text{ for }\: -1 \leq x \leq 1sin(sin−1(x))=x for −1≤x≤1
  • cos⁡(cos⁡−1(x))=x  for  0≤x≤1\cos(\cos^{-1}(x)) = x \:\text{ for }\: 0 \leq x \leq 1cos(cos−1(x))=x for 0≤x≤1
  • tan⁡(tan⁡−1(x))=x  for  −∞<x<∞\tan(\tan^{-1}(x)) = x \:\text{ for }\: -\infty < x < \inftytan(tan−1(x))=x for −∞<x<∞

For all trigonometric functions, the composition of the function and its inverse is equal to the input value. This is the same for the next case where the composition is of the form f−1(f(x))f^{-1}(f(x))f−1(f(x))...

  • sin⁡−1(sin⁡(x))=x  only for  −π2≤x≤π2\sin^{-1}(\sin(x)) = x \:\text{ only for }\: -\dfrac{\pi}{2} \leq x \leq \dfrac{\pi}{2}sin−1(sin(x))=x only for −2π​≤x≤2π​
  • cos⁡−1(cos⁡(x))=x  only for  0≤x≤π\cos^{-1}(\cos(x)) = x \:\text{ only for }\: 0 \leq x \leq \picos−1(cos(x))=x only for 0≤x≤π
  • tan⁡−1(tan⁡(x))=x  only for  −π2<x<π2\tan^{-1}(\tan(x)) = x \:\text{ only for }\: -\dfrac{\pi}{2} < x < \dfrac{\pi}{2}tan−1(tan(x))=x only for −2π​<x<2π​
note

We have to be careful with composition because sin⁡−1(sin⁡(x))≠x\sin^{-1}(\sin(x)) \neq xsin−1(sin(x))=x because this is not true for all values of xxx. The domain has to be restricted to the range of the inverse for the composition to be equal to the input value. This is the same for the other trigonometric functions as well.

Different Functions​

The third case is when the composition is of the form f−1(g(x))f^{-1}(g(x))f−1(g(x)) where both functions are trigonometric functions but not the same functions. For special values of xxx, we can exactly evaluate the inner function and then the outer function to find the value of the composition. This is a good approach however we can find a more general approach using a right triangle.

Consider the relation between two acute angles of a right triangle where one is θ\thetaθ making the other one π2−θ\dfrac{\pi}{2} - \theta2π​−θ. Using the fact that cos⁡θ=bc=sin⁡(π2−θ)\cos \theta = \dfrac{b}{c} = \sin(\dfrac{\pi}{2} - \theta)cosθ=cb​=sin(2π​−θ), we can deduce that sin⁡−1(cos⁡θ)=π2−θ\sin^{-1}(\cos \theta) = \dfrac{\pi}{2} - \thetasin−1(cosθ)=2π​−θ. Similarly, based on sin⁡θ=ac=cos⁡(π2−θ)\sin \theta = \dfrac{a}{c} = \cos(\dfrac{\pi}{2} - \theta)sinθ=ca​=cos(2π​−θ), we can deduce that cos⁡−1(sin⁡θ)=π2−θ\cos^{-1}(\sin \theta) = \dfrac{\pi}{2} - \thetacos−1(sinθ)=2π​−θ. This means that the composition of the sine and cosine functions is equal to the complementary angle of the original angle.

Using these new relations, given functions of the form sin⁡−1(cos⁡(x))\sin^{-1}(\cos(x))sin−1(cos(x)) or cos⁡−1(sin⁡(x))\cos^{-1}(\sin(x))cos−1(sin(x)), we can evaluate the composition as follows:

  1. If x∈[0,π]x \in [0, \pi]x∈[0,π], then sin⁡−1(cos⁡(x))=π2−x\sin^{-1}(\cos(x)) = \dfrac{\pi}{2} - xsin−1(cos(x))=2π​−x.
  2. If x∉[0,π]x \notin [0, \pi]x∈/[0,π], then find another angle yyy in [0,π][0, \pi][0,π] such that cos⁡(y)=cos⁡(x)\cos(y) = \cos(x)cos(y)=cos(x). Then follow the first step.
  3. If x∈[−π2,π2]x \in [-\dfrac{\pi}{2}, \dfrac{\pi}{2}]x∈[−2π​,2π​], then cos⁡−1(sin⁡(x))=π2−x\cos^{-1}(\sin(x)) = \dfrac{\pi}{2} - xcos−1(sin(x))=2π​−x.
  4. If x∉[−π2,π2]x \notin [-\dfrac{\pi}{2}, \dfrac{\pi}{2}]x∈/[−2π​,2π​], then find another angle yyy in [−π2,π2][-\dfrac{\pi}{2}, \dfrac{\pi}{2}][−2π​,2π​] such that sin⁡(y)=sin⁡(x)\sin(y) = \sin(x)sin(y)=sin(x). Then follow the third step.

Example‾\underline{Example}Example​

Evaluate cos⁡−1(sin⁡(−11π4))\cos^{-1}(\sin(-\dfrac{11\pi}{4}))cos−1(sin(−411π​)).

For the form cos⁡−1(sin⁡(x))\cos^{-1}(\sin(x))cos−1(sin(x)), we need xxx to be in the range [−π2,π2][-\dfrac{\pi}{2}, \dfrac{\pi}{2}][−2π​,2π​]. In this case, the angle is −11π4-\dfrac{11\pi}{4}−411π​ which is not in the range and so we need to find another angle yyy such that sin⁡(y)=sin⁡(−11π4)\sin(y) = \sin(-\dfrac{11\pi}{4})sin(y)=sin(−411π​) and y∈[−π2,π2]y \in [-\dfrac{\pi}{2}, \dfrac{\pi}{2}]y∈[−2π​,2π​].

The angle −11π4-\dfrac{11\pi}{4}−411π​ is coterminal with −3π4-\dfrac{3\pi}{4}−43π​ which is in the third quadrant. Next, the reference angle of 3π4\dfrac{3\pi}{4}43π​ is π4\dfrac{\pi}{4}4π​. Combining this with the fact that sine is negative in the third quadrant, we can find the angle y=−π4y = -\dfrac{\pi}{4}y=−4π​.

Now that we have an angle in the range of [−π2,π2][-\dfrac{\pi}{2}, \dfrac{\pi}{2}][−2π​,2π​], we can use the relation cos⁡−1(sin⁡(x))=π2−x\cos^{-1}(\sin(x)) = \dfrac{\pi}{2} - xcos−1(sin(x))=2π​−x to find the value of the composition.

cos⁡−1(sin⁡(−11π4))=π2−x=π2−(−π4)=π2+π4=3π4\cos^{-1}(\sin(-\dfrac{11\pi}{4})) = \dfrac{\pi}{2} - x = \dfrac{\pi}{2} - (-\dfrac{\pi}{4}) = \dfrac{\pi}{2} + \dfrac{\pi}{4} = \dfrac{3\pi}{4}cos−1(sin(−411π​))=2π​−x=2π​−(−4π​)=2π​+4π​=43π​.

The solution is cos⁡−1(sin⁡(−11π4))=3π4\cos^{-1}(\sin(-\dfrac{11\pi}{4})) = \dfrac{3\pi}{4}cos−1(sin(−411π​))=43π​.

Finally, the last form of composition is f(g−1(x))f(g^{-1}(x))f(g−1(x)) where both functions are trigonometric functions but not the same functions. In this case, we can use the pythagorean identities like sin⁡2(x)+cos⁡2(x)=1\sin^2(x) + \cos^2(x) = 1sin2(x)+cos2(x)=1 and our knowledge with trigonometric functions to find the value of the composition.

Starting with the inside, we can state there exists an angle θ\thetaθ such that sin⁡−1(79)=θ\sin^{-1}(\dfrac{7}{9}) = \thetasin−1(97​)=θ which means sin⁡θ=79\sin \theta = \dfrac{7}{9}sinθ=97​. Using the pythagorean identity, we can find the cosine value of the angle θ\thetaθ.

sin⁡2θ+cos⁡2θ=1(79)2+cos⁡2θ=14981+cos⁡2θ=1cos⁡2θ=1−4981cos⁡2θ=3281cos⁡θ=429\begin{array}{llllllllll} \sin^2 \theta + \cos^2 \theta &=& 1 \\[0.5em] (\frac{7}{9})^2 + \cos^2 \theta &=& 1 \\[0.5em] \frac{49}{81} + \cos^2 \theta &=& 1 \\[0.5em] \cos^2 \theta &=& 1 - \frac{49}{81} \\[0.5em] \cos^2 \theta &=& \frac{32}{81} \\[0.5em] \cos \theta &=& \frac{4\sqrt{2}}{9} \\[0.5em] \end{array}sin2θ+cos2θ(97​)2+cos2θ8149​+cos2θcos2θcos2θcosθ​======​1111−8149​8132​942​​​
