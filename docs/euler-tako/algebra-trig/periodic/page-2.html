Other Trigonometric Functions

Besides cosine and sine, there are other trigonometric functions that are derived from them. These functions also have periodic properties and allow us to model real-world phenomena which makes it vital to be able to graph them as well.

note

The tangent function is an odd function because sine is an odd function and cosine is an even function. This causes tangent to be odd because the tangent function is equivalent to the quotient of an odd function and an even function which always results in an odd function. In fact, any function that is the quotient of an odd function and an even function is an odd function.

General Equation​

Similar to the sine and cosine functions, the tangent function can also be described by a general equation. The general equation of the tangent function is y=Atan⁡(Bx−C)+Dy = A\tan(Bx - C) + Dy=Atan(Bx−C)+D where AAA, BBB, CCC, and DDD are all constants.

The features of the graph of y=Atan⁡(Bx−C)+Dy = A\tan(Bx - C) + Dy=Atan(Bx−C)+D are as follows...

  • The stretching factor is ∣A∣|A|∣A∣.
  • The period is π∣B∣\dfrac{\pi}{|B|}∣B∣π​.
  • The domain is x≠CB+π2kx \neq \dfrac{C}{B} + \dfrac{\pi}{2}kx=BC​+2π​k where kkk is an integer.
  • The range is (−∞,∞)(-\infty, \infty)(−∞,∞).
  • The vertical asymptotes occur at x=CB+π2kx = \dfrac{C}{B} + \dfrac{\pi}{2}kx=BC​+2π​k where kkk is an odd integer.
  • There is no horizontal asymptote.
  • There is no amplitude.

Graphing Tangent​

Given the function y=Atan⁡(Bx−C)+Dy = A \tan(Bx - C) + Dy=Atan(Bx−C)+D, we can sketch the graph of one period using the following steps...

  1. Verify the function is in the form y=Atan⁡(Bx−C)+Dy = A \tan(Bx - C) + Dy=Atan(Bx−C)+D.
  2. Identify the stretching/compressing factor, ∣A∣|A|∣A∣.
  3. Identify BBB and use it to find the period, P=π∣B∣P = \dfrac{\pi}{|B|}P=∣B∣π​.
  4. Identify CCC and use it to find the phase shift, CB\dfrac{C}{B}BC​.
  5. Sketch the vertical asymptotes, which occur at x=CB+π2∣B∣kx = \dfrac{C}{B} + \dfrac{\pi}{2|B|}kx=BC​+2∣B∣π​k where kkk is an odd integer.
  6. For AB>0AB > 0AB>0, the graph approaches the left asymptote at negative output values and the right asymptote at positive output values. On the other hand, for AB<0AB < 0AB<0, the graph approaches the left asymptote at positive output values and the right asymptote at negative output values.
  7. Obtain the reference points at (P4,A)(\dfrac{P}{4}, A)(4P​,A), (0,0)(0, 0)(0,0), and (−P4,−A)(-\dfrac{P}{4}, -A)(−4P​,−A).
  8. Shift the reference points to the right by CB\dfrac{C}{B}BC​ and up by DDD.
  9. Sketch the graph using the reference points and the asymptotes.

Secant Function​

The secant function can also be defined as sec⁡θ=1cos⁡θ\sec \theta = \dfrac{1}{\cos \theta}secθ=cosθ1​. Considering the fact that the function is undefined when cosine is equal to 000, the secant function has vertical asymptotes at π2\dfrac{\pi}{2}2π​, 3π2\dfrac{3\pi}{2}23π​, etc. Also note that since the cosine function will never be less than −1-1−1 or greater than 111, the secant function will never be between −1-1−1 and 111 as cosine is in the denominator.

Putting this all together, this gives us the graph of the secant function...

note

Since cosine is an even function, secant is also an even function. This is because the reciprocal of any even function is also an even function.

General Equation​

The general equation of the secant function is y=Asec⁡(Bx−C)+Dy = A\sec(Bx - C) + Dy=Asec(Bx−C)+D where AAA, BBB, CCC, and DDD are all constants.

The features of the graph of y=Asec⁡(Bx−C)+Dy = A\sec(Bx - C) + Dy=Asec(Bx−C)+D are as follows...

  • The stretching factor is ∣A∣|A|∣A∣.
  • The period is 2π∣B∣\dfrac{2\pi}{|B|}∣B∣2π​.
  • The domain is x≠CB+π2∣B∣kx \neq \dfrac{C}{B} + \dfrac{\pi}{2|B|}kx=BC​+2∣B∣π​k, where kkk is an odd integer.
  • The range is (−∞,−∣A∣+D]∪[∣A∣+D,∞)(-\infty, -|A| + D] \cup [|A| + D, \infty)(−∞,−∣A∣+D]∪[∣A∣+D,∞).
  • The vertical asymptotes occur at x=CB+π2∣B∣kx = \dfrac{C}{B} + \dfrac{\pi}{2|B|}kx=BC​+2∣B∣π​k where kkk is an odd integer.
  • There is no horizontal asymptote.
  • There is no amplitude.

Graphing Secant​

Given the function y=Asec⁡(Bx−C)+Dy = A\sec(Bx - C) + Dy=Asec(Bx−C)+D, we can sketch the graph of one period using the following steps...

  1. Verify the function is in the form y=Asec⁡(Bx−C)+Dy = A\sec(Bx - C) + Dy=Asec(Bx−C)+D.
  2. Identify the stretching/compressing factor, ∣A∣|A|∣A∣.
  3. Identify BBB and use it to find the period, P=2π∣B∣P = \dfrac{2\pi}{|B|}P=∣B∣2π​.
  4. Identify CCC and use it to find the phase shift, CB\dfrac{C}{B}BC​.
  5. Sketch the vertical asymptotes, which occur at x=CB+π2∣B∣kx = \dfrac{C}{B} + \dfrac{\pi}{2|B|}kx=BC​+2∣B∣π​k where kkk is an odd integer.
  6. Sketch the graph of y=Acos⁡(Bx−C)+Dy = A\cos(Bx - C) + Dy=Acos(Bx−C)+D.
  7. Use the reciprocal relationship between y=cos⁡xy = \cos xy=cosx and y=sec⁡xy = \sec xy=secx to sketch the graph.

The stretching factor is ∣A∣=6|A| = 6∣A∣=6.

The constant B=4B = 4B=4 so the period is P=2π∣B∣=2π4=π2P = \dfrac{2\pi}{|B|} = \dfrac{2\pi}{4} = \dfrac{\pi}{2}P=∣B∣2π​=42π​=2π​.

Cosecant Function​

The cosecant function can also be defined as csc⁡θ=1sin⁡θ\csc \theta = \dfrac{1}{\sin \theta}cscθ=sinθ1​. Similar to the secant function, the cosecant function is undefined when sine is equal to 000 which gives us vertical asymptotes at 000, π\piπ, 2π2\pi2π, etc. Also, since the sine function will never be less than −1-1−1 or greater than 111, the cosecant function will never be between −1-1−1 and 111 as sine is in the denominator.

This gives us the graph of the cosecant function...

note

Since sine is an odd function, cosecant is also an odd function. This is because the reciprocal of any odd function is also an odd function.

General Equation​

The general equation of the cosecant function is y=Acsc⁡(Bx−C)+Dy = A\csc(Bx - C) + Dy=Acsc(Bx−C)+D where AAA, BBB, CCC, and DDD are all constants.

The features of the graph of y=Acsc⁡(Bx−C)+Dy = A\csc(Bx - C) + Dy=Acsc(Bx−C)+D are as follows...

  • The stretching factor is ∣A∣|A|∣A∣.
  • The period is 2π∣B∣\dfrac{2\pi}{|B|}∣B∣2π​.
  • The domain is x≠CB+π∣B∣kx \neq \dfrac{C}{B} + \dfrac{\pi}{|B|}kx=BC​+∣B∣π​k, where kkk is an integer.
  • The range is (−∞,−∣A∣+D]∪[∣A∣+D,∞)(-\infty, -|A| + D] \cup [|A| + D, \infty)(−∞,−∣A∣+D]∪[∣A∣+D,∞).
  • The vertical asymptotes occur at x=CB+π∣B∣kx = \dfrac{C}{B} + \dfrac{\pi}{|B|}kx=BC​+∣B∣π​k where kkk is an integer.
  • There is no horizontal asymptote.
  • There is no amplitude.

Graphing Cosecant​

Given the function y=Acsc⁡(Bx−C)+Dy = A\csc(Bx - C) + Dy=Acsc(Bx−C)+D, we can sketch the graph of one period using the following steps...

  1. Verify the function is in the form y=Acsc⁡(Bx−C)+Dy = A\csc(Bx - C) + Dy=Acsc(Bx−C)+D.
  2. Identify the stretching/compressing factor, ∣A∣|A|∣A∣.
  3. Identify BBB and use it to find the period, P=2π∣B∣P = \dfrac{2\pi}{|B|}P=∣B∣2π​.
  4. Identify CCC and use it to find the phase shift, CB\dfrac{C}{B}BC​.
  5. Sketch the vertical asymptotes, which occur at x=CB+π∣B∣kx = \dfrac{C}{B} + \dfrac{\pi}{|B|}kx=BC​+∣B∣π​k where kkk is an integer.
  6. Sketch the graph of y=Asin⁡(Bx−C)+Dy = A\sin(Bx - C) + Dy=Asin(Bx−C)+D.
  7. Use the reciprocal relationship between y=sin⁡xy = \sin xy=sinx and y=csc⁡xy = \csc xy=cscx to sketch the graph.

Cotangent Function​

Finally, the cotangent function can be defined as cot⁡θ=1tan⁡θ\cot \theta = \dfrac{1}{\tan \theta}cotθ=tanθ1​. Considering the fact that the function is undefined when tangent is equal to 000, the cotangent function has vertical asymptotes at 000, π\piπ, 2π2\pi2π, etc. Also, since the output of the tangent function is all real numbers, the cotangent function will also have all real numbers as its output.

This gives us the graph of the cotangent function...

note

Since tangent is an odd function, cotangent is also an odd function. This is because the reciprocal of any odd function is also an odd function.

General Equation​

The general equation of the cotangent function is y=Acot⁡(Bx−C)+Dy = A\cot(Bx - C) + Dy=Acot(Bx−C)+D where AAA, BBB, CCC, and DDD are all constants.

The features of the graph of y=Acot⁡(Bx−C)+Dy = A\cot(Bx - C) + Dy=Acot(Bx−C)+D are as follows...

  • The stretching factor is ∣A∣|A|∣A∣.
  • The period is π∣B∣\dfrac{\pi}{|B|}∣B∣π​.
  • The domain is x≠CB+π∣B∣kx \neq \dfrac{C}{B} + \dfrac{\pi}{|B|}kx=BC​+∣B∣π​k, where kkk is an integer.
  • The range is (−∞,∞)(-\infty, \infty)(−∞,∞).
  • The vertical asymptotes occur at x=CB+π∣B∣kx = \dfrac{C}{B} + \dfrac{\pi}{|B|}kx=BC​+∣B∣π​k where kkk is an integer.
  • There is no horizontal asymptote.
  • There is no amplitude.
