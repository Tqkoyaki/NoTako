Radicals

Radicals are the opposite operations to exponent like how we can undo a multiplication by division, we can undo an exponent through a radical and vice versa.

Square Roots​

Some important terms include...

  1. The principal square root of aaa is the nonnegative number that, when multiplied by itself, equals aaa.
  2. A radical is the symbol (\sqrt{}​) which is used to denote a root.
  3. The radicand is the expression written inside a root like aaa in a\sqrt{a}a​.
  4. A radical expression is an expression which includes a radical term.
_________ for Simplifying Square RootsDefinition
Product Ruleab=a×b\sqrt{ab} = \sqrt{a} \times \sqrt{b}ab​=a​×b​
Quotient Ruleab=ab\sqrt{\frac{a}{b}} = \frac{\sqrt{a}}{\sqrt{b}}ba​​=b​a​​
note

We can simplify roots like 300\sqrt{300}300​ through these rules. Through the product rule this expression can be rewritten as 100×3=100×3\sqrt{100 \times 3} = \sqrt{100} \times \sqrt{3}100×3​=100​×3​. This gives us the simpliest form being 10310\sqrt{3}103​.

We can only add or subtract terms with the same radical. For example, the expression 55+355\sqrt{5} + 3\sqrt{5}55​+35​ can be simplified to 858\sqrt{5}85​. Also, expressions like 300+53\sqrt{300} + 5\sqrt{3}300​+53​ can be simplified too only when we simplify 300\sqrt{300}300​ to include the same radical. This gives us the expression 103+5310\sqrt{3} + 5\sqrt{3}103​+53​ which simplifies to 15315\sqrt{3}153​.

Rationalizing Denominators​

If a radical expression is written in simplest form then it cannot contain a radical in the denominator. The process of removing these radicals from the denominators of fractions is called rationalizing the denominator.

For single term denominators in the form bcb\sqrt{c}bc​, multiply the fraction by cc\frac{\sqrt{c}}{\sqrt{c}}c​c​​. It must be noted that bbb can be equal to 111.

For denominators that contain a sum of a rational and an irrational term in the form a+bca + b\sqrt{c}a+bc​, then we use the conjugate which is a−bca - b\sqrt{c}a−bc​. This means we multiply the fraction with a−bca−bc\frac{a - b\sqrt{c}}{a - b\sqrt{c}}a−bc​a−bc​​.

Finally, for denominators that contain a difference of a rational and an irrational term in the form a−bca - b\sqrt{c}a−bc​, then we use the conjugate which is a+bca + b\sqrt{c}a+bc​. This means we multiply the fraction with a+bca+bc\frac{a + b\sqrt{c}}{a + b\sqrt{c}}a+bc​a+bc​​.

note

We are multiplying these fractions by fractions that are equivalent to 111 when we are simplifying. The identity property of multiplication shows us we can do this without changing the value of the original fraction. Also note, we obtained the conjugates by changing the sign of the radical portion of the denominator.

Rational Roots​

  1. If aaa is a real number with at least one nnnth root, then the principal nnnth root of aaa is the number with the same sign as aaa that, when raised to the nnnth power, equals aaa. This root is written as an\sqrt[n]{a}na​.
  2. The index of the radical in a radical expression is the value of nnn in an\sqrt[n]{a}na​.
note

The expression a\sqrt{a}a​ is equivalent to writing a2\sqrt[2]{a}2a​.

Rational exponents are another way to express principal nnnth roots. The general form is amna^{\frac{m}{n}}anm​ which is equivalent to (an)m(\sqrt[n]{a})^m(na​)m and amn\sqrt[n]{a^m}nam​. All three expressions are equivalent to each other.
