Inverse Functions

There are various instances where we have functions that we want to use but we have information about the output and we want to find the input. This is where inverse functions come in. They allow us to rewrite a function in terms of its output rather than its input.

Inverse Functions​

Given the functions fff and ggg, they are inverses of each other if every coordinate (a,b)(a, b)(a,b) on the graph of fff cooresponds to the coordinate (b,a)(b, a)(b,a) on the graph of ggg. Note that not all functions have inverses because only functions that are one-to-one have inverses. If the function fff is not one-to-one, then the function ggg would not be a function as it would have multiple outputs for a single input.

We can find the inverse of any one-to-one function by following these steps...

  1. Replace f(x)f(x)f(x) with yyy
  2. Interchange xxx and yyy
  3. Solve for yyy, and rename the function f−1(x)f^{-1}(x)f−1(x)

Example‾\underline{Example}Example​

Find the inverse of the function f(x)=2x+3f(x) = 2x + 3f(x)=2x+3

Replacing f(x)f(x)f(x) with yyy and interchanging xxx and yyy gives us x=2y+3x = 2y + 3x=2y+3. We can now solve for yyy...

2x=2y+32y=x−32y=12x−32\begin{array}{l} \phantom{2}x = 2y + 3 \\ 2y = x - 3 \\ \phantom{2}y = \frac{1}{2}x - \frac{3}{2} \end{array}2x=2y+32y=x−32y=21​x−23​​

The inverse of f(x)f(x)f(x) is f−1(x)=12x−32f^{-1}(x) = \frac{1}{2}x - \frac{3}{2}f−1(x)=21​x−23​.

Domain​

The domain of a function is the set of all possible inputs that the function can take. The domain of the inverse function is the range of the original function. This fact gives us the following properties...

  1. f−1(f(x))=xf^-1(f(x)) = xf−1(f(x))=x for all xxx in the domain of fff.
  2. f(f−1(x))=xf(f^{-1}(x)) = xf(f−1(x))=x for all xxx in the domain of f−1f^{-1}f−1.

We can use these properties to verify if two functions are inverses of each other because if these properties hold true for any two functions, then they are inverses of each other.

Domain Restrictions​

As previously stated, for a function to have an inverse, it must be one-to-one. A function that has an inverse is called an invertible function.

There are functions like f(x)=x2f(x) = x^2f(x)=x2 that are not invertible but can be made invertible by restricting the domain. For the case of f(x)=x2f(x) = x^2f(x)=x2, we can restrict the domain to x≥0x \geq 0x≥0 or x≤0x \leq 0x≤0 to make the function invertible.

Example‾\underline{Example}Example​

Find the inverse of the function f(x)=(x−4)2f(x) = (x - 4)^2f(x)=(x−4)2 where x≥4x \geq 4x≥4.

The domain of f(x)f(x)f(x) makes it invertible because it restricts the quadratic function to only the right side of the axis of symmetry.

Replacing f(x)f(x)f(x) with yyy and interchanging xxx and yyy gives us x=(y−4)2x = (y - 4)^2x=(y−4)2. We can now solve for yyy...

x=(y−4)2→x=y−4→y=x+4x = (y - 4)^2 \to \sqrt{x} = y - 4 \to y = \sqrt{x} + 4x=(y−4)2→x​=y−4→y=x​+4

Note that we ignored −x+4-\sqrt{x} + 4−x​+4 because of our domain restriction. So the inverse of f(x)f(x)f(x) is f−1(x)=x+4f^{-1}(x) = \sqrt{x} + 4f−1(x)=x​+4.

Radical Functions​

Often times, the function we want to find the inverse of is a radical function. To find the inverse of a radical function, we can follow the same steps as before. However, after solving for yyy, we need to restrict the domain of the inverse function to the range of the original function.

Example‾\underline{Example}Example​

For y=12x2−32y = \frac{1}{2}x^2 - \frac{3}{2}y=21​x2−23​ to be the inverse of f(x)f(x)f(x), we need to restrict the domain. The domain of f−1(x)f^{-1}(x)f−1(x) is the range of f(x)f(x)f(x). Rational Functions​

When finding the inverse of rational functions, we often have multiple instances of xxx variables in the equation which we interchange with yyy. To solve for yyy in these situations, we need to use the distributive property to isolate yyy...

Example‾\underline{Example}Example​

Find the inverse of the function f(x)=x+3x−2f(x) = \frac{x + 3}{x - 2}f(x)=x−2x+3​.

Replacing f(x)f(x)f(x) with yyy and interchanging xxx and yyy gives us x=y+3y−2x = \frac{y + 3}{y - 2}x=y−2y+3​. We can now solve for yyy...

The inverse of f(x)f(x)f(x) is f−1(x)=2x+3x−1f^{-1}(x) = \frac{2x + 3}{x - 1}f−1(x)=x−12x+3​.
