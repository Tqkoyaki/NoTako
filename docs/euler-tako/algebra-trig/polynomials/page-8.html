Variation

There are various equations that describe the relationship between two variables where one quantity changes proportionally to another. We refer to these relationships as variations.

Direct Variation​

If xxx and yyy are related by an equation of the form y=kxny = kx^ny=kxn then we state that the relationship is direct variation which means yyy varies directly with, or is proportional to, the nnnth power of xxx. In these relationships, kkk is called the constant of variation and it is a non-zero constant ratio where k=yxnk = \frac{y}{x^n}k=xny​. This constant is what defines the relationship between the two variables.

Given a description of a direction variation problem, we can solve for an unknown by...

  1. Identifying the input and output variables.
  2. Determining the constant of variation which is either given or can be found using the ratio k=yxnk = \frac{y}{x^n}k=xny​.
  3. Use the constant of variation to write an equation for the relationship.
  4. Solve for the unknown.

Example‾\underline{Example}Example​

The quantity yyy varies directly with the square of xxx. If y=24y = 24y=24 when x=3x = 3x=3, find yyy when xxx is 444.

The input is defined by xxx, the output is defined as yyy, and the power of the function is 222. We still need to find the constant of variation using the ratio:

k=yxn=2432=249=83k = \frac{y}{x^n} = \frac{24}{3^2} = \frac{24}{9} = \frac{8}{3}k=xny​=3224​=924​=38​

With all the information, we can write the equation for the relationship as y=83x2y = \frac{8}{3}x^2y=38​x2. We can use this equation to solve for yyy when x=4x = 4x=4 by substituting the value into the equation:

y=83(4)2=83(16)=1283y = \frac{8}{3}(4)^2 = \frac{8}{3}(16) = \frac{128}{3}y=38​(4)2=38​(16)=3128​

Therefore, when x=4x = 4x=4, y=1283y = \frac{128}{3}y=3128​.

Inverse Variation​

If xxx and yyy are related by an equation of the form y=kxny = \frac{k}{x^n}y=xnk​ then we state that the relationship is inverse variation which means yyy varies inversely proportional with the nnnth power of xxx. The constant of variation, kkk, is a non-zero constant ratio where k=xnyk = x^nyk=xny.

We can solve for an unknown in an inverse variation problem the same way we do for direct variation problems. The only difference is that the ratio we use to find the constant of variation is k=xnyk = x^nyk=xny.

Example‾\underline{Example}Example​

A quantity yyy varies inversely with the square of xxx. If y=8y = 8y=8 where x=3x = 3x=3, find yyy when xxx is 444.

The input is defined by xxx, the output is defined as yyy, and the power of the function is 222. We still need to find the constant of variation using the ratio:

k=xny=32(8)=9(8)=72k = x^ny = 3^2(8) = 9(8) = 72k=xny=32(8)=9(8)=72

With all the information, we can write the equation for the relationship as y=72x2y = \frac{72}{x^2}y=x272​. We can use this equation to solve for yyy when x=4x = 4x=4 by substituting the value into the equation:

y=7242=7216=92y = \frac{72}{4^2} = \frac{72}{16} = \frac{9}{2}y=4272​=1672​=29​

Therefore, when x=4x = 4x=4, y=92y = \frac{9}{2}y=29​.

Joint Variation​

The final type of variation is joint variation which occurs when a variable varies directly or inversely with multiple variables. For example, if xxx varies directly with both yyy and zzz, then x=kyzx = kyzx=kyz. This can get more general like if xxx varies directly with yyy and inversely with zzz, we have x=kyzx = \frac{ky}{z}x=zky​.

note

We only use one constant of variation in a joint variation equation.

Example‾\underline{Example}Example​

A quantity xxx varies directly with the square of yyy and inversely with zzz. If x=40x = 40x=40 when y=4y = 4y=4 and z=2z = 2z=2, find xxx when y=10y = 10y=10 and z=25z = 25z=25.

The value of xxx is dependent on both yyy and zzz. The relationship between xxx and yyy is a direct variation which means x=ky2x = ky^2x=ky2 and the relationship between xxx and zzz is an inverse variation which means x=kzx = \frac{k}{z}x=zk​. Combining these two relationships, we get x=ky2zx = \frac{ky^2}{z}x=zky2​ which only has one constant of variation because it is a joint variation.

We can use the equation x=ky2zx = \frac{ky^2}{z}x=zky2​ to solve for kkk by substituting the values of xxx, yyy, and zzz:

40=k(4)22→40=16k2→40=8k→k=540 = \frac{k(4)^2}{2} \to 40 = \frac{16k}{2} \to 40 = 8k \to k = 540=2k(4)2​→40=216k​→40=8k→k=5

The relationship between xxx, yyy, and zzz is x=5y2zx = \frac{5y^2}{z}x=z5y2​. We can use this equation to solve for xxx when y=10y = 10y=10 and z=25z = 25z=25 by substituting the values into the equation:

x=5y2z=5(10)225=5(100)25=50025=20x = \frac{5y^2}{z} = \frac{5(10)^2}{25} = \frac{5(100)}{25} = \frac{500}{25} = 20x=z5y2​=255(10)2​=255(100)​=25500​=20

Therefore, when y=10y = 10y=10 and z=25z = 25z=25, x=20x = 20x=20.
