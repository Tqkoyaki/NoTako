Zeros of Polynomial Functions

We have methods to find the zeros of linear and quadratic functions. We can take this a step further with methods that use polynomial division and allow us to find the zeros of any polynomial function regardless of the degree.

Remainder Theorem​

The Remainder Theorem states that if a polynomial f(x)f(x)f(x) is divided by x−kx - kx−k, then the remainder is the value f(k)f(k)f(k).

We can arrive at the remainder theorem from the division theorem. The division algorithm states that given a polynomial dividend f(x)f(x)f(x) and a non-zero polynomial divisor d(x)d(x)d(x), there exist unique polynomials q(x)q(x)q(x) and r(x)r(x)r(x) such that f(x)=d(x)q(x)+r(x)f(x) = d(x)q(x) + r(x)f(x)=d(x)q(x)+r(x) and r(x)r(x)r(x) is less than the degree of d(x)d(x)d(x). Factor Theorem​

The Factor Theorem states that kkk is a zero of f(x)f(x)f(x) if and only if (x−k)(x - k)(x−k) is a factor of f(x)f(x)f(x). This means that if f(x)f(x)f(x) divided by (x−k)(x - k)(x−k) results in a remainder of 000 then kkk is a zero of f(x)f(x)f(x).

Rational Zero Theorem​

The Rational Zero Theorem states that, if the polynomial f(x)=anxn+an−1xn−1+...+a1x+a0f(x) = a_nx^n + a_{n - 1}x^{n - 1} + ... + a_1x + a_0f(x)=an​xn+an−1​xn−1+...+a1​x+a0​ has integer coefficients and an≠0a_n \neq 0an​=0, then every rational zero of f(x)f(x)f(x) has the form pq\frac{p}{q}qp​ where ppp is a factor of the constant term a0a_0a0​ and qqq is a factor of the leading coefficient ana_nan​. This means we can determine all the possible rational zeros of f(x)f(x)f(x) by finding all the possible values of pq\frac{p}{q}qp​.

note

Both the Factor Theorem and the Rational Zero Theorem are theorems built on the concepts of the Remainder Theorem.

Zeros of Polynomial Functions​

Given a polynomial function fff, we can find the zeros using the following steps...

  1. Use the Rational Zero Theorem to list all possible rational zeros of fff.
  2. Use synthetic division to evaluate a given possible zero by dividing the candidate into the polynomial. If the remainder is 000, the candidate is a zero. If the remainder is not a 000, discard the candidate.
  3. Repeat step two using the quotient found with synthetic division. Repeat till the quotient is a quadratic.
  4. Find the zeros of the quadratic function.

Example‾\underline{Example}Example​

Find the zeros of f(x)=4x3−3x−1f(x) = 4x^3 - 3x - 1f(x)=4x3−3x−1

The factors of the leading coefficient are ±1\pm 1±1, ±2\pm 2±2, and ±4\pm 4±4. The only factor of the constant is ±1\pm 1±1. This means pq=±11\frac{p}{q} = \pm\frac{1}{1}qp​=±11​, ±12\pm\frac{1}{2}±21​ and ±14\pm\frac{1}{4}±41​.

Fundemental Theorem of Algebra​

The Fundemental Theorem of Algebra states that, if f(x)f(x)f(x) is a polynomial of degree n>0n > 0n>0, then f(x)f(x)f(x) has at least one complex zero. This theorem can be used to also argue that, if f(x)f(x)f(x) is a polynomial of degree n>0n > 0n>0, and aaa is a non-zero real number, then f(x)f(x)f(x) has exactly nnn linear factors and f(x)=a(x−c1)(x−c2)...(x−cn)f(x) = a(x - c_1)(x - c_2)...(x - c_n)f(x)=a(x−c1​)(x−c2​)...(x−cn​) where c1c_1c1​, c2c_2c2​, ..., cnc_ncn​ are complex numbers. Therefore, f(x)f(x)f(x) has nnn roots if we allow for multiplicities.

note

All real numbers are a subset of complex numbers and that means a real number can be the complex zero for a given function. However, remember that not all complex numbers are real numbers.

Complex Conjugate Theorem​

The Linear Factorization Theorem states that a polynomial function will have the same number of factors as its degree, and each factor will be in the form (x−c)(x - c)(x−c), where ccc is a complex number. Building from this, the Complex Conjugate Theorem states that if the polynomial function fff has real coefficients and a complex zero in the form a+bia + bia+bi, then the complex conjugate of the zero, a−bia - bia−bi, is also a zero.

Descartes' Rule of Signs​

The Descartes' Rule of Signs is a method used to determine the possible numbers of positive and negative real zeros for any polynomial function. According to this rule, if we let f(x)=anxn+an−1xn−1+...+a1x+a0f(x) = a_nx^n + a_{n - 1}x^{n - 1} + ... + a_1x + a_0f(x)=an​xn+an−1​xn−1+...+a1​x+a0​ be a polynomial function with real coefficients:

  1. The number of positive real zeros is either equal to the number of sign changes of f(x)f(x)f(x) or is less than the number of sign changes by an even integer.
  2. The number of negative real zeros is either equal to the number of sign changes of f(−x)f(-x)f(−x) or is less than the number of sign changes by an even integer.

Example‾\underline{Example}Example​

Use Descartes' Rule of Signs to determine the maximum possible numbers of positive and negative real zeros for f(x)=2x4−10x3+11x2−15x+12f(x) = 2x^4 - 10x^3 + 11x^2 - 15x + 12f(x)=2x4−10x3+11x2−15x+12.

There are 444 sign changes for f(x)f(x)f(x) which means there can only be 444, 222, or 000 positive real zeros.

To find the negative real zeros, we need to solve for f(−x)f(-x)f(−x) first...

f(−x)=2(−x)4−10(−x)3+11(−x)2−15(−x)+12f(−x)=2x4+10x3+11x2+15x+12\begin{array}{l} f(-x) = 2(-x)^4 - 10(-x)^3 + 11(-x)^2 - 15(-x) + 12 \\ \phantom{f(-x)} = 2x^4 + 10x^3 + 11x^2 + 15x + 12 \end{array}f(−x)=2(−x)4−10(−x)3+11(−x)2−15(−x)+12f(−x)=2x4+10x3+11x2+15x+12​

There are no sign changes in f(−x)f(-x)f(−x) which means there can only be 000 negative real zeros.
