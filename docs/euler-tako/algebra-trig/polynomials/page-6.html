Rational Functions

A rational function is a function that can be written as a quotient of two polynomial functions P(x)P(x)P(x) and Q(x)Q(x)Q(x) which means f(x)=P(x)Q(x)=qpxp+ap−1xp−1+...+a1x+a0bqxq+bq−1xq−1+...+b1x+b0f(x) = \frac{P(x)}{Q(x)} = \frac{q_px^p + a_{p - 1}x^{p - 1} + ... + a_1x + a_0}{b_qx^q + b_{q - 1}x^{q - 1} + ... + b_1x + b_0}f(x)=Q(x)P(x)​=bq​xq+bq−1​xq−1+...+b1​x+b0​qp​xp+ap−1​xp−1+...+a1​x+a0​​ where Q(x)≠0Q(x) \neq 0Q(x)=0. These types of functions are useful as many problems involving rates and concentrations often involve rational functions.

Domain​

The domain of a rational function includes all real numbers except those that cause the denominator to equal zero.

Vertical Asymptote​

A vertical asymptote of a graph is a vertical line x=ax = ax=a where the graph tends toward positive or negative infinity as the input approaches aaa from either the left or the right. We write as x→a−,f(x)→±∞x \to a^-, f(x) \to \pm \inftyx→a−,f(x)→±∞ or x→a+,f(x)→±∞x \to a^+, f(x) \to \pm \inftyx→a+,f(x)→±∞.

Given a rational function, we can find any vertical asymptotes of its graph by...

  1. Factor the numerator and denominator.
  2. Reduce the function by cancelling common factors in the numerator and denominator.
  3. All the values that cause the denominator to be zero in this simplified version are where the vertical asymptotes occur.

Example‾\underline{Example}Example​

Find the vertical asymptotes of the graph of f(x)=x−2x2+2x−8f(x) = \frac{x - 2}{x^2 + 2x - 8}f(x)=x2+2x−8x−2​.

First we factor the function and then reduce it if possible...

f(x)=x−2x2+2x−8=(x−2)(x−2)(x+4)=(x−2)(x−2)(x+4)=0(x+4)f(x) = \frac{x - 2}{x^2 + 2x - 8} = \frac{(x - 2)}{(x - 2)(x + 4)} = \frac{\bcancel{(x - 2)}}{\bcancel{(x - 2)}{(x + 4)}} = \frac{0}{(x + 4)}f(x)=x2+2x−8x−2​=(x−2)(x+4)(x−2)​=(x−2)​(x+4)(x−2)​​=(x+4)0​

The denominator for the reduced form is x+4x + 4x+4 which is zero only if x=−4x = -4x=−4.

The only vertical asymptote of the graph is x=−4x = -4x=−4.

Multiplicity​

The multiplicity of a vertical asymptote is the number of times the factor is repeated in the denominator. This is important because it determines the behavior of the graph at a vertical asymptote.

  1. If the multiplicity is even, the graph will tend towards the same direction on both sides of the asymptote.
  2. If the multiplicity is odd, the graph will tend towards infinity on one side and negative infinity on the other side of the asymptote.

Removable Discontinuity​

A removable discontinuity occurs in the graph of a rational function at x=ax = ax=a. These are single points where the graph is not defined. We often refer to these discountinuity as holes as indicate them by a open circle in the graph.

We can find these discontinuities through the cancelled out factors during reduction. Given a rational function, the steps are...

  1. Factor the numerator and denominator.
  2. Reduce the function by cancelling common factors in the numerator and denominator.
  3. All the values that cause the denominator to be zero in this simplified version are where the vertical asymtotes occur.
  4. All the values that cause the non-simplified version to be zero but are not vertical asymptotes are where the removable discontinuities occur.

Example‾\underline{Example}Example​

Find the removable discontinuities of the graph of f(x)=x−2x2+2x−8f(x) = \frac{x - 2}{x^2 + 2x - 8}f(x)=x2+2x−8x−2​.

First we factor the function and then reduce it if possible...

f(x)=x−2x2+2x−8=(x−2)(x−2)(x+4)=(x−2)(x−2)(x+4)=0(x+4)f(x) = \frac{x - 2}{x^2 + 2x - 8} = \frac{(x - 2)}{(x - 2)(x + 4)} = \frac{\bcancel{(x - 2)}}{\bcancel{(x - 2)}{(x + 4)}} = \frac{0}{(x + 4)}f(x)=x2+2x−8x−2​=(x−2)(x+4)(x−2)​=(x−2)​(x+4)(x−2)​​=(x+4)0​

The only factor that was cancelled out was (x−2)(x - 2)(x−2) and x−2=0x - 2 = 0x−2=0 when x=2x = 2x=2.

The only removable discontinuity of the graph is at x=2x = 2x=2.

note

A graph will continually approach a given asymptote or discontinuity but will never meet it at any finite distance.

Horizontal Asymptote​

A horizontal asymptote of a graph is a horizontal line y=by = by=b where the graph approaches the line as the inputs increase or decreases without bound. We write as x→∞x \to \inftyx→∞ or x→−∞x \to -\inftyx→−∞, f(x)→bf(x) \to bf(x)→b.

The horizontal asymptote of a rational function can determined through the degrees of the numerator and denominator...

  1. If the degree of the numerator is less than the degree of the denominator, then there is a horizontal asymptote at y=0y = 0y=0.
  2. If the degree of the numerator is greater than the degree of the denominator by one, then there is no horizontal asymptote.
  3. If the degree of the numerator is equal to the degree of the denominator, then the horizontal asymptote is at the ratio of leading coefficients.

Example‾\underline{Example}Example​

Find the horizontal asymptote of f(x)=6x2−10x2x2+5xf(x) = \frac{6x^2 - 10x}{2x^2 + 5x}f(x)=2x2+5x6x2−10x​.

The degree of the numerator is 222 and the degree of the denominator is 222. The degrees are equal and so the horizontal asymptote is at the ratio of leading coefficients which is 62\frac{6}{2}26​.

This means the horizontal asymptote is at y=3y = 3y=3.

The degree of the numerator is one greater than the denominator which means we can find the slant asymptote by dividing the numerator by the denominator. Graphing​

Given a rational function, we can graph it by following these steps...

  1. Evaluate the function at x=0x = 0x=0 to find the yyy-intercept.
  2. Factor the numerator and denominator.
  3. For factors in the numerator not common to the denominator, determine where each factor of the numerator is zero to find the xxx-intecepts.
  4. Find the multiplicities of the xxx-intercepts to determine the behavior of the graph at those points.
  5. For factors in the denominator that are not common to the numerator, find the vertical asymptotes by finding the zeros of those factors.
  6. Note the multiplicity of the vertical asymptotes.
  7. For factors in the denominator that are common to the numerator, find the removable discontinuities by finding the zeros of those factors.
  8. Compare the degrees of the numerator and the denominator to determine the horizontal or slant asymptotes.
  9. Sketch the graph using all the information found.

Example‾\underline{Example}Example​

Sketch the graph of f(x)=(x+2)2(x−2)2(x−1)2(x−3)f(x) = \frac{(x + 2)^2(x - 2)}{2(x - 1)^2(x - 3)}f(x)=2(x−1)2(x−3)(x+2)2(x−2)​.

The rational function is already factored, so lets find the xxx-intercepts and their multiplicities. The first xxx-intercept is at x=−2x = -2x=−2 with a multiplicity of 222 because (x+2)2=0(x + 2)^2 = 0(x+2)2=0. The second xxx-intercept is at x=2x = 2x=2 with a multiplicity of 111 because (x−2)=0(x - 2) = 0(x−2)=0.

The next step is to find the vertical asymptotes. The first vertical asymptote is at x=1x = 1x=1 because (x−1)2=0(x - 1)^2 = 0(x−1)2=0. The second vertical asymptote is at x=3x = 3x=3 because (x−3)=0(x - 3) = 0(x−3)=0.

The multiplicity of x=1x = 1x=1 is 222 meaning the graph will tend towards the same direction on both sides of the asymptote. The multiplicity of x=3x = 3x=3 is 111 meaning the graph will tend towards negative infinity on one side and positive infinity on the other side of the asymptote.

There are no removable discontinuities because all factors in the numerator are not in the denominator.

The final step before sketching the graph is to find the horizontal asymptote. The degree of the numerator is 333 and the degree of the denominator is 333. The degrees are equal which means the horizontal asymptote is at the ratio of leading coefficients which is 12\frac{1}{2}21​.
