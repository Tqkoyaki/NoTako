Trigonometric Equations

We have explored the trigonometric functions and their properties in the previous sections. However, those are just tools to help us work with equations that involve trigonometric functions. By utilizing the properties of trigonometric functions and algebraic manipulation, we can solve a variety of equations, including ones that model real-world phenomena.

Linear Equations​

Oftentimes, we will solve a trigonometric equation over a specific interval. However, just as often, we will be interested in all possible solutions which may cause us to have an infinite number of solutions as trigonometric functions are periodic.

We know that trigonometric functions repeat their values at regular intervals called the period so we could add or subtract PkPkPk to a solution of the equation where PPP is the period of the function and kkk is any integer. This would give us a general solution to the equation.

Example‾\underline{Example}Example​

Find all the solutions to the following linear equation: 2sin⁡x+1=02 \sin x + 1 = 02sinx+1=0

We can start by isolating the sine function...

2sin⁡x+1=02sin⁡x=−1sin⁡x=−12\begin{array}{cccccccc} 2 \sin x + 1 &=& 0 \\[0.5em] 2 \sin x &=& -1 \\[0.5em] \sin x &=& -\frac{1}{2} \\[0.5em] \end{array}2sinx+12sinxsinx​===​0−1−21​​

Using the unit circle, we can find that sin⁡x=−12\sin x = -\frac{1}{2}sinx=−21​ at x=7π6x = \frac{7\pi}{6}x=67π​ and x=11π6x = \frac{11\pi}{6}x=611π​.

However, we are looking for all solutions to the equation so we need to add the period of the sine function to both solutions using x=Pkx = Pkx=Pk where P=2πP = 2\piP=2π and kkk is any integer.

The solutions are x=7π6+2πkx = \frac{7\pi}{6} + 2\pi kx=67π​+2πk and x=11π6+2πkx = \frac{11\pi}{6} + 2\pi kx=611π​+2πk where kkk is any integer.

note

Just like rational equations, the domain of a trigonometric function must be considered before assuming that any solution is valid because we could have any number of solutions including no solution at all depending on the domain.

Single Function Equations​

When working with equations that involve only one trigonometric function, their solutions typically involve using algebraic techniques and the unit circle. However, when these equations involve trigonometric functions other than sine and cosine, we typically try to use trigonometric properties to solve the equation for sine or cosine because they are convenient to work with.

Example‾\underline{Example}Example​

Find all solutions in the interval [0,4π)[0, 4\pi)[0,4π) to the following equation: csc⁡θ=−2\csc \theta = -2cscθ=−2.

We can rewrite cosecant in terms of sine and then isolate the sine function...

csc⁡θ=−21sin⁡θ=−21=−2sin⁡θsin⁡θ=−12\begin{array}{cccccccc} \csc \theta &=& -2 \\[0.5em] \dfrac{1}{\sin \theta} &=& -2 \\[1em] 1 &=& -2 \sin \theta \\[0.5em] \sin \theta &=& -\frac{1}{2} \\[0.5em] \end{array}cscθsinθ1​1sinθ​====​−2−2−2sinθ−21​​

Using the unit circle, we can find that sin⁡θ=−12\sin \theta = -\dfrac{1}{2}sinθ=−21​ at θ=7π6\theta = \dfrac{7\pi}{6}θ=67π​ and θ=11π6\theta = \dfrac{11\pi}{6}θ=611π​.

Using the periodicity of sine, we can find all solutions in the interval [0,4π)[0, 4\pi)[0,4π) by adding the period of sine to each solution: θ=7π6,11π6,7π6+2π,11π6+2π\theta = \dfrac{7\pi}{6}, \dfrac{11\pi}{6}, \dfrac{7\pi}{6} + 2\pi, \dfrac{11\pi}{6} + 2\piθ=67π​,611π​,67π​+2π,611π​+2π. All other solutions are outside of the interval [0,4π)[0, 4\pi)[0,4π).

The solutions are θ=7π6,11π6,19π6,23π6\theta = \dfrac{7\pi}{6}, \dfrac{11\pi}{6}, \dfrac{19\pi}{6}, \dfrac{23\pi}{6}θ=67π​,611π​,619π​,623π​.

Tangent and Cotangent Equations​

The use of just sine or cosine tends to not be possible all the time with tangent and cotangent functions so we try to solve for tangent and use tangents period of π\piπ to find any additional solutions.

Example‾\underline{Example}Example​

Find all solutions in the interval [0,2π)[0, 2\pi)[0,2π) to the following equation: tan⁡(θ−π2)=1\tan (\theta - \dfrac{\pi}{2}) = 1tan(θ−2π​)=1.

Using the unit circle, we can find that tan⁡θ=1\tan \theta = 1tanθ=1 at θ=π4\theta = \dfrac{\pi}{4}θ=4π​. However, the angle we are looking for is θ−π2\theta - \dfrac{\pi}{2}θ−2π​ which we can find using the equation θ−π2=π4\theta - \dfrac{\pi}{2} = \dfrac{\pi}{4}θ−2π​=4π​. This gives us θ=π4+π2=3π4\theta = \dfrac{\pi}{4} + \dfrac{\pi}{2} = \dfrac{3\pi}{4}θ=4π​+2π​=43π​.

We can use the fact that the period of tangent is π\piπ to find the general solution: θ=3π4+πk\theta = \dfrac{3\pi}{4} + \pi kθ=43π​+πk where kkk is any integer.

Using this general solution, we can find all solutions in the interval [0,2π)[0, 2\pi)[0,2π) by substituting k=0k = 0k=0 and k=1k = 1k=1. Every other solution will be outside of the interval.

This gives us the solutions θ=3π4\theta = \dfrac{3\pi}{4}θ=43π​ and θ=7π4\theta = \dfrac{7\pi}{4}θ=47π​.

Quadratic Form​

The quadratic form is a pattern we can find in a multitude of equations including ones that involve trigonmetric functions. If there is only one trigonometric function represented and one of the terms is squared, then we can think about the equation as a quadratic equation. This means we can replace the trigonometric function with a variable such as xxx or uuu and then solve the equation as we would a quadratic equation. Once we have solved for the variable, we can substitute back in the trigonometric function to find the solutions.

Example‾\underline{Example}Example​

Solve the quadratic equation 2cos⁡2θ+cos⁡θ=02 \cos^2{\theta} + \cos{\theta} = 02cos2θ+cosθ=0.

We can substitute u=cos⁡θu = \cos{\theta}u=cosθ to get the following equation: 2u2+u=02u^2 + u = 02u2+u=0. We can now solve for uuu using algebraic techniques.

2u2+u=0u(2u+1)=0u=0or2u+1=0u=0oru=−12\begin{array}{cccccccc} 2u^2 + u &=& 0 \\[0.5em] u(2u + 1) &=& 0 \\[0.5em] u = 0 \quad &\text{or}& \quad 2u + 1 = 0 \\[0.5em] u = 0 \quad &\text{or}& \quad u = -\dfrac{1}{2} \\[0.5em] \end{array}2u2+uu(2u+1)u=0u=0​==oror​002u+1=0u=−21​​

We can now substitute back in for uuu to get the following equations: cos⁡θ=0\cos{\theta} = 0cosθ=0 and cos⁡θ=−12\cos{\theta} = -\dfrac{1}{2}cosθ=−21​.

Solving for cos⁡θ=0\cos{\theta} = 0cosθ=0 gives us the solutions θ=π2+πk\theta = \dfrac{\pi}{2} + \pi kθ=2π​+πk where kkk is any integer.

Solving for cos⁡θ=−12\cos{\theta} = -\dfrac{1}{2}cosθ=−21​ gives us the solutions θ=2π3+2πk\theta = \dfrac{2\pi}{3} + 2\pi kθ=32π​+2πk and θ=4π3+2πk\theta = \dfrac{4\pi}{3} + 2\pi kθ=34π​+2πk where kkk is any integer.

The solutions in the interval [0,2π)[0, 2\pi)[0,2π) are θ=π2,2π3,4π3,3π2\theta = \dfrac{\pi}{2}, \dfrac{2\pi}{3}, \dfrac{4\pi}{3}, \dfrac{3\pi}{2}θ=2π​,32π​,34π​,23π​.

Fundemental Identities​

Algebraic techniques are valuable for working with trigonometric equations but we also have a variety of powerful identities that are just as valuable because they make solving equations simpler. Once we identity that a portion of the equation matches any side of an identity, we can replace it with the other side to make the equation easier to work with. By doing this, we can solve a plethora of equations that involve trigonometric functions.

Example‾\underline{Example}Example​

Use identities to solve the trigonometric equation over the interval [0,2π)[0, 2\pi)[0,2π): cos⁡xcos⁡(2x)+sin⁡xsin⁡(2x)=32\cos x \cos{(2x)} + \sin x \sin{(2x)} = \dfrac{\sqrt{3}}{2}cosxcos(2x)+sinxsin(2x)=23​​.

The left side of the equation, cos⁡xcos⁡(2x)+sin⁡xsin⁡(2x)\cos x \cos{(2x)} + \sin x \sin{(2x)}cosxcos(2x)+sinxsin(2x), matches the cosine of a sum identity: cos⁡(a+b)=cos⁡acos⁡b−sin⁡asin⁡b\cos{(a + b)} = \cos a \cos b - \sin a \sin bcos(a+b)=cosacosb−sinasinb. We can use this identity to simplify and solve the equation...

cos⁡xcos⁡(2x)+sin⁡xsin⁡(2x)=32cos⁡(x−2x)=32cos⁡(−x)=32\begin{array}{cccccccc} \cos x \cos{(2x)} + \sin x \sin{(2x)} &=& \dfrac{\sqrt{3}}{2} \\[1em] \cos{(x - 2x)} &=& \dfrac{\sqrt{3}}{2} \\[1em] \cos{(-x)} &=& \dfrac{\sqrt{3}}{2} \\[1em] \end{array}cosxcos(2x)+sinxsin(2x)cos(x−2x)cos(−x)​===​23​​23​​23​​​

We can simplify further by using the negative angle identity where cos⁡(−x)=cos⁡x\cos{(-x)} = \cos xcos(−x)=cosx to get the following equation: cos⁡x=32\cos x = \dfrac{\sqrt{3}}{2}cosx=23​​.

Using the unit circle, we can find that cos⁡x=32\cos x = \dfrac{\sqrt{3}}{2}cosx=23​​ at x=π6x = \dfrac{\pi}{6}x=6π​ and x=11π6x = \dfrac{11\pi}{6}x=611π​.

The solutions in the interval [0,2π)[0, 2\pi)[0,2π) are x=π6x = \dfrac{\pi}{6}x=6π​ and x=11π6x = \dfrac{11\pi}{6}x=611π​.

Multiple Angles​

Sometimes it is not possible to solve a trigonmetric equation with identities that have multiple angles such as sin⁡(2x)\sin(2x)sin(2x) or cos⁡(3x)\cos(3x)cos(3x). In these cases, we can use our fundemental knowledge of trigonometry to analyze the equation and find a solution. For example, we know that sin⁡(2x)\sin(2x)sin(2x) is a horizontal compression of sin⁡x\sin xsinx by a factor of 222. This means there are twice as many solutions in the same interval because the function y=sin⁡(2x)y = \sin(2x)y=sin(2x) will complete 222 cycles at the same time as y=sin⁡xy = \sin xy=sinx completes one cycle. This is valuable information because it allows us to understand how many solutions to expect and gives us another way to analyze the equation which in turn helps us solve the equation.

Example‾\underline{Example}Example​

Solve exactly: cos⁡(2x)=12\cos(2x) = \dfrac{1}{2}cos(2x)=21​ on [0,2π)[0,2\pi)[0,2π).

The trigonometric function cos⁡(2x)\cos(2x)cos(2x) is a horizontal compression of the cosine function by a factor of 222 which means that the function will complete 222 cycles in the same interval as the cosine function. This means we can expect 222 times as many solutions in the interval [0,2π)[0, 2\pi)[0,2π).

Using the unit circle, we can find that cos⁡x=12\cos x = \dfrac{1}{2}cosx=21​ at x=π3x = \dfrac{\pi}{3}x=3π​ and x=5π3x = \dfrac{5\pi}{3}x=35π​. As we are compressing the function by a factor of 222, we can find the solutions for cos⁡(2x)=12\cos(2x) = \dfrac{1}{2}cos(2x)=21​ by dividing the solutions by 222. This gives us x=π6x = \dfrac{\pi}{6}x=6π​ and x=5π6x = \dfrac{5\pi}{6}x=65π​.

We have 222 solutions in the interval [0,2π)[0, 2\pi)[0,2π) but we are expecting twice as many solutions compared to y=cos⁡xy = \cos xy=cosx. This means we need to add the period of the function to each solution to find the other solutions in the interval [0,2π)[0, 2\pi)[0,2π).

2x=π3+2πk2x=5π3+2πk2x=π3+2π(1)2x=5π3+2π(1)2x=7π32x=11π3x=7π6x=11π6\begin{array}{cccccccc} 2x &=& \dfrac{\pi}{3} + 2\pi k && 2x &=& \dfrac{5\pi}{3} + 2\pi k \\[1em] 2x &=& \dfrac{\pi}{3} + 2\pi (1) && 2x &=& \dfrac{5\pi}{3} + 2\pi (1) \\[1em] 2x &=& \dfrac{7\pi}{3} && 2x &=& \dfrac{11\pi}{3} \\[1em] x &=& \dfrac{7\pi}{6} && x &=& \dfrac{11\pi}{6} \\[1em] \end{array}2x2x2xx​====​3π​+2πk3π​+2π(1)37π​67π​​​2x2x2xx​====​35π​+2πk35π​+2π(1)311π​611π​​

Both x=7π6x = \dfrac{7\pi}{6}x=67π​ and x=11π6x = \dfrac{11\pi}{6}x=611π​ are inside the interval [0,2π)[0, 2\pi)[0,2π) so we can keep them as solutions. If we try any other value for kkk, we will get a solution outside of the interval.

So, the solutions are x=π6,5π6,7π6,11π6x = \dfrac{\pi}{6}, \dfrac{5\pi}{6}, \dfrac{7\pi}{6}, \dfrac{11\pi}{6}x=6π​,65π​,67π​,611π​.

When we solve a problem similar to the form of sin⁡(nx)=c\sin(nx) = csin(nx)=c, we must go around the unit circle nnn times.

Right Triangles​

Finally, we can use all of our techniques to solve trigonmetric equations with the properties of right triangles and the Pythagorean Theorem. This is useful for solving models that involve right triangles as countless real-world applications can be modelled using these right triangles.

Example‾\underline{Example}Example​

One of the cables that anchors the center of the London Eye Ferris wheel to the ground must be replaced. The center of the Ferris wheel is 69.569.569.5 meters above the ground, and the second anchor on the ground is 232323 meters from the base of the Ferris wheel. Approximately how long is the cable, and what is the angle of elevation (from ground up to the center of the Ferris wheel)?

We can find the length of the cable using the Pythagorean Theorem. We know the length of both of the sides which we can use to find the length of the hypotenuse...

c2=a2+b2c2=69.52+232c2=69.52+529c2=4830.25+529c2=5359.25c=5359.25c≈73.2 meters\begin{array}{cccccccc} c^2 &=& a^2 + b^2 \\[0.5em] c^2 &=& 69.5^2 + 23^2 \\[0.5em] c^2 &=& 69.5^2 + 529 \\[0.5em] c^2 &=& 4830.25 + 529 \\[0.5em] c^2 &=& 5359.25 \\[0.5em] c &=& \sqrt{5359.25} \\[0.5em] c &\approx& 73.2 \text{ meters} \\[0.5em] \end{array}c2c2c2c2c2cc​======≈​a2+b269.52+23269.52+5294830.25+5295359.255359.25​73.2 meters​

We can find the angle of elevation using the tangent function. We know the length of the opposite side and the adjacent side so we can use tan⁡θ=oppositeadjacent\tan \theta = \dfrac{opposite}{adjacent}tanθ=adjacentopposite​ to find the angle of elevation...

tan⁡θ=oppositeadjacenttan⁡θ=69.523θ=tan⁡−1(69.523)θ≈71.69∘\begin{array}{cccccccc} \tan \theta &=& \dfrac{opposite}{adjacent} \\[1em] \tan \theta &=& \dfrac{69.5}{23} \\[1em] \theta &=& \tan^{-1}(\dfrac{69.5}{23}) \\[1em] \theta &\approx& 71.69^{\circ} \\[1em] \end{array}tanθtanθθθ​===≈​adjacentopposite​2369.5​tan−1(2369.5​)71.69∘​

The length of the cable is approximately 73.273.273.2 meters and the angle of elevation is approximately 71.69∘71.69^{\circ}71.69∘.
