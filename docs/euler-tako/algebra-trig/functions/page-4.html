Function Composition

Assume that we have a function that defines the relationship between aaa and bbb and another function that expresses the relationship between bbb and ccc. We can use function composition to create a brand new function using these functions to express a relationship between aaa and ccc. This is one of the main ways to combine functions together.

Composition of Functions​

As discussed before, another way to combine functions is through function composition. Given the functions f(x)f(x)f(x) and g(x)g(x)g(x), the function composition is (f∘g)(x)=f(g(x))(f \circ g)(x) = f(g(x))(f∘g)(x)=f(g(x)). The left side of the equation is read as fff composed with ggg at xxx, and the right side of the equation is read as fff of ggg of xxx.

The way (f∘g)(x)(f \circ g)(x)(f∘g)(x) works is that the xxx is the input of ggg and the output of ggg becomes the input of fff. So, to evaluate (f∘g)(x)(f \circ g)(x)(f∘g)(x) for a specific value of xxx, we substitute the value into g(x)g(x)g(x) and solve for g(x)g(x)g(x). After this we substitute the resulting value of g(x)g(x)g(x) into f(x)f(x)f(x) and solve for f(x)f(x)f(x). This is the solution for (f∘g)(x)(f \circ g)(x)(f∘g)(x).

On the other hand, to evaluate (f∘g)(x)(f \circ g)(x)(f∘g)(x) for all values of xxx we substitute the expression for g(x)g(x)g(x) into f(x)f(x)f(x) and simplify.

Domain​

The domain of f(g(x))f(g(x))f(g(x)) is the set of inputs xxx in the domain of ggg for which g(x)g(x)g(x) is in the domain of fff.
