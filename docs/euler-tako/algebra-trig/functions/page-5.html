Function Transformation

In mathematics, we have systematic ways to alter the basic functions to build new functions out of them. This allows us to have better understanding of how to change functions and how operations on a function impact the look and behavior of the said function.

Vertical and Horizontal Shift​

One basic transformation involves shifting the entire graph of a function up, down, left or right. Given a function f(x)f(x)f(x)...

  1. The vertical shift shifts f(x)f(x)f(x) up or down. Let g(x)=f(x)+kg(x) = f(x) + kg(x)=f(x)+k where kkk is a constant, this new function is a vertical shift of f(x)f(x)f(x) by kkk units. If kkk is positive, the graph will shift up. If kkk is negative, the graph will shift down.
  2. The horizontal shift shifts f(x)f(x)f(x) left or right. Let g(x)=f(x−h)g(x) = f(x - h)g(x)=f(x−h) where hhh is a constant, this new function is a horizontal shift of f(x)f(x)f(x) by hhh units. If hhh is positive, the graph will shift right. If hhh is negative, the graph will shift left.

Putting these concepts together, g(x)=f(x−h)+kg(x) = f(x - h) + kg(x)=f(x−h)+k is a new function that shifts f(x)f(x)f(x) vertically by kkk units and horizontally by hhh units. An example of this is...

Reflection​

Another transformation that can be applied to a function is a reflection over the xxx or yyy axis. Given a function f(x)f(x)f(x)...

  1. The vertical reflection reflects f(x)f(x)f(x) over the xxx-axis. The function g(x)g(x)g(x) is a reflection of f(x)f(x)f(x) over the xxx-axis if g(x)=−f(x)g(x) = -f(x)g(x)=−f(x).
  2. The horizontal reflection reflects f(x)f(x)f(x) over the yyy-axis. The function g(x)g(x)g(x) is a reflection of f(x)f(x)f(x) over the yyy-axis if g(x)=f(−x)g(x) = f(-x)g(x)=f(−x).

For vertical reflection, we multiply all the outputs by −1-1−1 of the original function. Finally, for the horizontal reflection, we multiply all the inputs by −1-1−1 of the original function.

Even and Odd Function​

Some functions are symmetrical meaning when we reflect the function, we result in the original graph. There are two types of these functions...

  1. An even function is a function whose graph is symmetric about the yyy-axis. This means for every xxx, f(x)=f(−x)f(x) = f(-x)f(x)=f(−x). An example of an even function is f(x)=x2f(x) = x^2f(x)=x2 because f(−x)=(−x)2=x2f(-x) = (-x)^2 = x^2f(−x)=(−x)2=x2.
  2. An odd function is a function whose graph is symmetric about the origin. This means for every xxx, f(x)=−f(−x)f(x) = -f(-x)f(x)=−f(−x).
