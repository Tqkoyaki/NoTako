Exponential Functions

So far we have worked with linear growth which refers to when the original value from the range increases by the same amount over equal increments found in the domain. In contrast, exponential growth refers to when the original value from the range increases by the same percentage over equal increments found in the domain. Due to percent change, exponential growth will grow rapidly as the xxx increases.

Some common terms encountered when dealing with exponential functions include...

  1. Percent change: a change based on a percent of the original amount.
  2. Exponential growth: an increase based on a constant multiplicative rate of change over equal increments of time, that is, a percent increase of the original amount over time.
  3. Exponential decay: a decrease based on a constant multiplicative rate of change over equal increments of time, that is, a percent decrease of the original amount over time.

Exponential Functions​

In general, for any real number xxx, an exponential function is a function with the form f(x)=abxf(x) = ab^xf(x)=abx where aaa is a non-zero real number indicating the initial value and bbb is any positive real number indicating the base such that b≠1b \neq 1b=1.

Exponential functions have quite a few properties that we can extract using the form...

  1. The domain of fff is all real numbers.
  2. The range of fff is all positive real numbers if a>0a > 0a>0 and the range of fff is all negative real numbers if a<0a < 0a<0.
  3. The yyy-intercept is (0,a)(0, a)(0,a).
  4. The horizontal asymptote is y=0y = 0y=0.
  5. The function models exponential growth if b>1b > 1b>1 and the function models exponential decay if 0<b<10 < b < 10<b<1.

The base bbb is restricted to positive values to ensure that the outputs will be real numbers. For example, if we let a=1a = 1a=1, b=−9b = -9b=−9 and x=12x = \frac{1}{2}x=21​. Then f(x)=f(12)=1(−9)12=−9f(x) = f(\frac{1}{2}) = 1(-9)^{\frac{1}{2}} = \sqrt{-9}f(x)=f(21​)=1(−9)21​=−9​, which is not a real number.

Just like with other functions, we can evaluate an exponential function for any xxx-value by substituting the value in the function and simplifying...

Finding Equations​

Given two data points, we can write an exponential model in the form of f(x)=a(b)xf(x) = a(b)^xf(x)=a(b)x. If one of the data points has the form (0,a)(0, a)(0,a), then aaa is the initial value. Using aaa, we can substitute the second point into the equation f(x)=a(b)xf(x) = a(b)^xf(x)=a(b)x to solve for bbb. On the other hand, if neither of the data points have the form (0,a)(0, a)(0,a), we can substitute both points into two equations with the form f(x)=a(b)xf(x) = a(b)^xf(x)=a(b)x. After this, we can solve the resulting system of two equations in two unknowns to find aaa and bbb. In both of these case, we want to use the value of aaa and bbb to write the function in the form f(x)=a(b)xf(x) = a(b)^xf(x)=a(b)x.

Let's look at the case where we have a point in the form (0,a)(0, a)(0,a)...

Compound Interest​

Compound interest refers to interest earned not only on the original value, but on the accumulated value of the account. This concept of compounding is often found with savings instruments in which earnings are continually reinvested like mutual funds and retirement accounts. The annual percentage rate (APR) of these accounts is the yearly interest rate earned by these account. We also refer to them this as the nominal rate because the term nominal is used when the compounding occurs more than once in year causing the effective interest rate to be greater than the nominal rate.

Compound interest can be calculated using the formula A(t)=P(1+rn)ntA(t) = P(1 + \frac{r}{n})^{nt}A(t)=P(1+nr​)nt where A(t)A(t)A(t) is the account value, PPP is the principal/starting amount, rrr is the APR expressed as a decimal, nnn is the number of compounding periods in one year, and ttt is the number of years.

  2. The interest rate is 12%12\%12% which represents the APR so r=0.12r = 0.12r=0.12.
  3. The interest is compounded weekly so n=52n = 52n=52.
  4. We are looking for the A(t)A(t)A(t) in 303030 years so t=30t = 30t=30 and we are solving for A(t)A(t)A(t).

Euler's Number​

The amount earned on an account increases as the compound frequency increases. This means as we increase the value of nnn and keep everything else the same in the compound interest formula, the value of A(t)A(t)A(t) should always increase.

Euler's number is a number which is represented by the letter eee and this represents the irrational number found by increasing nnn without bound for (1+1n)n(1 + \frac{1}{n})^n(1+n1​)n. This number approximates to e≈2.718282e \approx 2.718282e≈2.718282 and is irrational because as previously stated, A(t)A(t)A(t) will never stop increasing as nnn increases. The value of eee is the value that the function (1+1n)n(1 + \frac{1}{n})^n(1+n1​)n converges to as it approaches infinity.

The constant was named by the Swiss mathematican Leonhard Euler who first investigated and discovered many of its properties and this number is used as a base for countless real-world exponential models.

Continuous Growth and Decay​

For most real-world phenomena, eee is used as the base for exponential functions and these models that use eee as the base are called continuous growth or decay models. For all real numbers ttt, and all positive numbers aaa and rrr, continuous growth or decay is represented by the formula A(t)=aertA(t) = ae^{rt}A(t)=aert where aaa is the initial value, rrr is the continuous growth rate per unit time, and ttt is the elapsed time. Note that if r>0r > 0r>0, then the formula represents continuous growth and if r<0r < 0r<0, then the formula represents continuous decay.
