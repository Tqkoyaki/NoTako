Graphing Exponential Functions

Exponential functions are used for many real-world applications in the sciences and business and they give us a method for making predictions. However, being able to make predictions off equations is not always enough. Being able to graph these functions can give us a visual representation of the data which is a powerful tool allowing us to arrive at more insightful conclusions.

Graphing​

In order to graph exponential functions, we need to be able to graph the parent function f(x)=bxf(x) = b^xf(x)=bx where b>0b > 0b>0 and b≠1b \neq 1b=1. This parent function has the following characteristics which are useful to note when graphing...

  1. It is a one-to-one function
  2. There is a horizontal asymptote at y=0y = 0y=0
  3. The domain is (−∞,∞)(-\infty, \infty)(−∞,∞) and the range is (0,∞)(0, \infty)(0,∞)
  4. There is no xxx-intercept but there is a yyy-intercept at (0,1)(0, 1)(0,1)
  5. The graph is increasing if b>1b > 1b>1 and decreasing if b<1b < 1b<1

Noting all these properties, we can graph an exponential function of the form f(x)=bxf(x) = b^xf(x)=bx by...

  1. Creating a table of points.
  2. Plotting at least 333 points from the table, including the yyy-intercept (0,1)(0, 1)(0,1).
  3. Drawing a smooth curve through the points.
  4. Stating the domain, range, and horizontal asymptote.

All functions of the form f(x)=bxf(x) = b^xf(x)=bx have the same domain, range, and horizontal asymptote. The domain is (−∞,∞)(-\infty, \infty)(−∞,∞), the range is (0,∞)(0, \infty)(0,∞) and the horizontal asymptote is y=0y = 0y=0.

note

For any exponential function of the form f(x)=abxf(x) = ab^xf(x)=abx, we call the base the constant ratio because as the input increases by 111, the output value will be the product of the base and the previous output, regardless of the value of aaa. For example, if f(x)=2xf(x) = 2^xf(x)=2x and f(2)=4f(2) = 4f(2)=4 then f(3)=f(2)∗b=4∗2=8f(3) = f(2) * b = 4 * 2 = 8f(3)=f(2)∗b=4∗2=8. This is useful when creating tables for graphing.

Transformations​

We can take the parent function f(x)=bxf(x) = b^xf(x)=bx and apply transformations to get any exponential function of the form f(x)=abxf(x) = ab^xf(x)=abx. This means with the ability to graph the parent function, we can graph any exponential function.

Vertical Shift​

The first transformation occurs when we add a constant ddd to the parent function f(x)=bxf(x) = b^xf(x)=bx, giving us f(x)=bx+df(x) = b^x + df(x)=bx+d. This shifts the graph vertically by ddd units in the same direction as the sign of ddd.

When we perform a vertical shift, the new properties are...

  1. The domain (−∞,∞)(-\infty, \infty)(−∞,∞) remains the same.
  2. The range is shifted by ddd units and so the new range is (d,∞)(d, \infty)(d,∞).
  3. The horizontal asymptote shifts by ddd units and so the new asymptote is y=dy = dy=d.
  4. The yyy-intercept shifts by ddd units and so the new yyy-intercept is (0,1+d)(0, 1 + d)(0,1+d).

Horizontal Shift​

The second transformation occurs when we add a constant ccc to the input of the parent function f(x)=bxf(x) = b^xf(x)=bx, giving us f(x)=bx+cf(x) = b^{x + c}f(x)=bx+c. This shifts the graph horizontally by ccc units in the opposite direction of the sign of ccc.

When we perform a vertical and horizontal shift, we get the equation of the form f(x)=bx+c+df(x) = b^{x + c} + df(x)=bx+c+d where the new properties are...

  1. The domain (−∞,∞)(-\infty, \infty)(−∞,∞) remains the same.
  2. The range is shifted by ddd units and so the new range is (d,∞)(d, \infty)(d,∞).
  3. The horizontal asymptote shifts by ddd units and so the new asymptote is y=dy = dy=d.
  4. The yyy-intercept changes to (0,bc+d)(0, b^c + d)(0,bc+d).

Stretch and Compression​

We can also stretch or compress the graph of the parent function by multiplying the parent function by a constant aaa, giving us f(x)=abxf(x) = ab^xf(x)=abx. If ∣a∣>1|a| > 1∣a∣>1, the graph is stretched vertically by a factor of aaa and if ∣a∣<1|a| < 1∣a∣<1, the graph is compressed vertically by a factor of aaa.

When we perform a stretch or compression, the new properties are...

  1. The domain (−∞,∞)(-\infty, \infty)(−∞,∞) remains the same.
  2. The range (0,∞)(0, \infty)(0,∞) remains the same.
  3. The horizontal asymptote y=0y = 0y=0 remains the same.
  4. The yyy-intercept changes to (0,a)(0, a)(0,a).

Reflection​

We can also reflect the graph of the parent function about the xxx-axis and the yyy-axis by multiplying the parent function by a negative constant.

To reflect about the xxx-axis, we use the function f(x)=−bxf(x) = -b^xf(x)=−bx and it has the following properties...

  1. The domain (−∞,∞)(-\infty, \infty)(−∞,∞) remains the same.
  2. The range is reflected about the xxx-axis and so the new range is (−∞,0)(-\infty, 0)(−∞,0).
  3. The horizontal asymptote is y=0y = 0y=0 and remains the same.
  4. The yyy-intercept is (0,−1)(0, -1)(0,−1).

To reflect about the yyy-axis, we use the function f(x)=b−x=(1b)xf(x) = b^{-x} = (\frac{1}{b})^xf(x)=b−x=(b1​)x and the properties stay the same as the parent function. Nothing changes except the graph is reflected about the yyy-axis.

General Function​

A translation of an exponential function has the form f(x)=abx+c+df(x) = ab^{x + c} + df(x)=abx+c+d where the parent function, y=bxy = b^xy=bx, b>1b > 1b>1, is shifted horizontally ccc units to the left, shifted vertically ddd units, stretched vertically by a factor of ∣a∣|a|∣a∣ if ∣a∣>0|a| > 0∣a∣>0, compressed vertically by a factor of ∣a∣|a|∣a∣ if ∣a∣<1|a| < 1∣a∣<1, and reflected about the xxx-axis if a<0a < 0a<0.
