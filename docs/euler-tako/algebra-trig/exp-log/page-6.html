Solving Functions

Exponential and Logarithmic functions are able to model a wide variety of real-world phenomena and help us solve for unknown variables in a variety of situations. We can use the definitions and laws of exponents and logarithms to solve and manipulate all sorts of equations.

Exponential Functions​

Exponential functions are any functions which have a variable in the exponent. Oftentimes, complexity arises when there are multiple terms with variables as exponents. We can use the laws of exponents to simplify these expressions and solve for unknown variables.

Common Bases​

For any algebraic expressions SSS and TTT, and any positive real number b≠1b \neq 1b=1, bS=bTb^S = b^TbS=bT if and only if S=TS = TS=T. In other words, when the bases are the same between two exponential expressions, the exponents must be equal in order for the expressions to be equal. This property is called the One-to-One Property of Exponents.

If we have an exponential equation that can be written in the form bS=bTb^S = b^TbS=bT, where SSS and TTT are algebraic expressions, we can solve for the unknowns by first writing the equation in this form. After this, we can use the one to one property to eliminate the bases and rewrite the equation as S=TS = TS=T. Finally, we can solve for the unknowns by solving the resulting equation.

Example‾\underline{Example}Example​

Solve 52x=5x+35^{2x} = 5^{x + 3}52x=5x+3.

The equation already has the same base and is in the form bS=bTb^S = b^TbS=bT. We can use the one-to-one property of exponents to eliminate the bases and solve for xxx...

52x=5x+32x=x+3x=3\begin{array}{llllllllll} 5^{2x} &=& 5^{x + 3} \\ 2x &=& x + 3 \\ x &=& 3 \\ \end{array}52x2xx​===​5x+3x+33​

The solution to the equation is x=3x = 3x=3.

Even when the bases are not the same, we can still solve for the unknowns by using the properties of exponents to rewrite the equation in a form where the bases are the same. Once each side of the equation has the same base, we can use our previous method to solve for the unknowns.

Example‾\underline{Example}Example​

Solve for 55x=55^{5x} = \sqrt{5}55x=5​

We can rewrite 5\sqrt{5}5​ as 5125^{\frac{1}{2}}521​ to give us a base of 555 on both sides of the equations. We can use the one-to-one property of exponents to eliminate the bases and solve for $x...

55x=555x=5125x=12x=110\begin{array}{llllllllll} 5^{5x} &=& \sqrt{5} \\ 5^{5x} &=& 5^{\frac{1}{2}} \\ 5x &=& \frac{1}{2} \\ x &=& \frac{1}{10} \\ \end{array}55x55x5xx​====​5​521​21​101​​

The solution to the equation is x=110x = \frac{1}{10}x=101​.

Different Bases​

Sometimes the terms cannot be rewritten with the same base. In these caeses, we can solve by taking the logarithm on each side of the equation since log⁡(a)=log⁡(b)\log(a) = \log(b)log(a)=log(b) is equivalent to a=ba = ba=b. Applying the logarithm to both sides of the equation is useful as it allows us to use the power rule to seperate the exponents from the bases making it easier to solve for the unknowns.

Given an exponential equation in which a common base cannot be found, we can solve for the unknowns by first applying the logarithm to both sides of the equation. If one of the terms has base 101010, we use the common logarithm. If none of the terms in the equation has base 101010, we use the natural logarithm. Next, we use the rules of logarithms to simplify the equation and solve for the unknowns.

Example‾\underline{Example}Example​

Solve for 2x=3x+12^x = 3^{x + 1}2x=3x+1.

The solution to the equation is x=ln⁡(3)ln⁡(23)x = \frac{\ln(3)}{\ln(\frac{2}{3})}x=ln(32​)ln(3)​.

Natural Bases​

One common type of exponential equations we use involve the natural base, eee, and come in the form of y=Aekty = Ae^{kt}y=Aekt. These equations constantly appear in nature, in mathematics, in science, in engineering, and in finance. Given an equation of the form y=Aekty = Ae^kty=Aekt, we can solve for ttt through the following steps...

  1. Divide both sides of the equation by AAA.
  2. Apply the natural logarithm of both sides to both sides of the equation.
  3. Divide both sides of the equation by kkk.

Example‾\underline{Example}Example​

Solve for ttt in the equation 3e0.5t=113e^{0.5t} = 113e0.5t=11.

We can solve for ttt by dividing both sides by 333 and then applying the natural logarithm to both sides...

The solution to the equation is t=2ln⁡(113)t = 2\ln(\frac{11}{3})t=2ln(311​).

Extraneous Solutions​

There are countless methods in mathematics that are used to solve equations but they introduce extraneous solutions which are solutions that are correct algebraically but do not satisfy the conditions of the original equation. One such situation arises when the logarithm is taken on both sides of the equation. In such cases, we must note that the argument of the logarithm must be positive and so if the number we are evaluating in a logarithm function is negative, there is no output.

Logarithmic Functions​

The inverse of an exponential function is a logarithmic function and we need to be able to solve and manipulate logarithmic equations as much as we do exponential equations. We can use the definition and properties of logarithms to simplify and solve logarithmic equations.

Definition of a Logarithm​

We typically see logarithmic equations in the form log⁡b(x)=y\log_b(x) = ylogb​(x)=y which is equivalent to the exponential equation by=xb^y = xby=x. We can make many of these logarithmic equations easier to solve by rewriting the equation in exponential form. For any algebraic expression SSS and real numbers bbb and ccc, where b>0b > 0b>0, b≠1b \neq 1b=1, log⁡b(S)=c\log_b(S) = clogb​(S)=c if and only if bc=Sb^c = Sbc=S.

Example‾\underline{Example}Example​

Solve for 2ln⁡(x+1)=102\ln(x + 1) = 102ln(x+1)=10.

This equation is not in the form log⁡b(x)=y\log_b(x) = ylogb​(x)=y but we can manipulate it to be that form. We can divide both sides by 222 to get ln⁡(x+1)=5\ln(x + 1) = 5ln(x+1)=5 and then rewrite the equation in exponential form to get e5=x+1e^5 = x + 1e5=x+1...

ln⁡(x+1)=5e5=x+1x=e5−1\begin{array}{llllllllll} \ln(x + 1) &=& 5 \\ e^5 &=& x + 1 \\ x &=& e^5 - 1 \\ \end{array}ln(x+1)e5x​===​5x+1e5−1​

The solution to the equation is x=e5−1x = e^5 - 1x=e5−1.

One to One Functions​

Just like with exponential functions, logarithmic functions are one-to-one functions. The one-to-one property of logarithms states that for any real numbers x>0x > 0x>0, S>0S > 0S>0, T>0T > 0T>0, and any positive real number bbb, where b≠1b \neq 1b=1, log⁡b(S)=logb(T)\log_b(S) = log_b(T)logb​(S)=logb​(T) if and only if S=TS = TS=T. This means that when the bases are the same between two logarithmic expressions, the arguments must be equal in order for the expressions to be equal.

Given a logarithmic equation in the form log⁡b(S)=log⁡b(T)\log_b(S) = \log_b(T)logb​(S)=logb​(T), we can solve for the unknowns by first writing the equation in this form. After this, we can use the one to one property to eliminate the bases and rewrite the equation as S=TS = TS=T. Finally, we can solve for the unknowns by solving the resulting equation.

Example‾\underline{Example}Example​

Solve for ln⁡(x2)=ln(1)\ln(x^2) = ln(1)ln(x2)=ln(1).

The equation is already in the form log⁡b(S)=log⁡b(T)\log_b(S) = \log_b(T)logb​(S)=logb​(T) so we can use the one-to-one property to eliminate the bases and solve for xxx...

ln⁡(x2)=ln⁡(1)x2=1x=±1\begin{array}{llllllllll} \ln(x^2) &=& \ln(1) \\ x^2 &=& 1 \\ x &=& \pm 1 \\ \end{array}ln(x2)x2x​===​ln(1)1±1​

The solution to the equation is x=±1x = \pm 1x=±1.

Half Life​

As previously mentioned, exponential functions and logarithmic functions can be used to model real-world phenomena. One such application is in the field of science where we can model the decay of a radioactive substance. The half-life of a substance is the time it takes for half of the unstable material in a sample of a radioactive substance to decay. We can use the following equivalent equations to solve for the half-life of a substance...
