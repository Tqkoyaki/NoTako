Unit Circle

Now that we have defined the trigonometric functions in terms of right triangles, we are ready to redefine them in terms of a unit circle. A unit circle is a circle centered at the origin with a radius of 111. It is divided into four quadrants using the xxx and yyy axes and labeled quadrants I, II, III, and IV in a counterclockwise direction.

The benefit of using unit circles is that for any angle θ\thetaθ, we can label the intersection of the terminal side and the unit circle by its coordinates (x,y)(x, y)(x,y) where the xxx-coordinate is always equal to cos⁡θ\cos \thetacosθ and the yyy-coordinate is always equal to sin⁡θ\sin \thetasinθ.

note

Given that we can find arc length using the formula s=rθs = r \thetas=rθ, there is an interesting property of the unit circle that states the arc length of a unit circle is equal to the radian measure of the angle. This is because the radius of the unit circle is 111 and so s=1×θ=θs = 1 \times \theta = \thetas=1×θ=θ.

Sine and Cosine​

As mentioned, the sine function of an angle θ\thetaθ is equal to the yyy-coordinate of the point where the corresponding angle intercepts the unit circle. Similarly, the cosine function of an angle θ\thetaθ is equal to the xxx-coordinate of the point where the corresponding angle intercepts the unit circle. This means if θ\thetaθ is a real number and a point (x,y)(x, y)(x,y) on the unit circle corresponds to a central angle θ\thetaθ, then sin⁡θ=y\sin \theta = ysinθ=y and cos⁡θ=x\cos \theta = xcosθ=x.

Example‾\underline{Example}Example​

Find cos⁡(90°)\cos(90\degree)cos(90°) and sin⁡(90°)\sin(90\degree)sin(90°).

All angles in the unit circle are measured in radians so we need to convert 90°90\degree90° to radians. Since 180°=π180\degree = \pi180°=π radians, we can find that 90°=π290\degree = \frac{\pi}{2}90°=2π​ radians.

Since π2\frac{\pi}{2}2π​ radians divided by 2π2\pi2π is 14\frac{1}{4}41​, we can see that π2\frac{\pi}{2}2π​ is equivalent to 14\frac{1}{4}41​ of the unit circle. If we move counterclockwise a quarter of the way from the positive xxx-axis (the point (1,0)(1, 0)(1,0)), we will reach the point (0,1)(0, 1)(0,1).

Since the xxx-coordinate of (0,1)(0, 1)(0,1) is 000, we can say that cos⁡π2=0\cos \frac{\pi}{2} = 0cos2π​=0. Similarly, since the yyy-coordinate of (0,1)(0, 1)(0,1) is 111, we can say that sin⁡π2=1\sin \frac{\pi}{2} = 1sin2π​=1.

Therefore, cos⁡90°=0\cos 90\degree = 0cos90°=0 and sin⁡90°=1\sin 90\degree = 1sin90°=1.

note

The domain of the sine and cosine functions are all real numbers because angles smaller than 000 and larger than 2π2\pi2π can still be graphed on the unit circle. However, the range of the sine and cosine functions are [−1,1][-1, 1][−1,1] because the xxx and yyy bounds of the unit circle are −1-1−1 and 111 due to the radius being 111.

Pythagorean Identity​

Now that we have defined the sine and cosine functions in terms of the unit circle, we are able to study its characteristics. Recall that the formula for a circle is x2+y2=r2x^2 + y^2 = r^2x2+y2=r2 and since the radius of the unit circle is 111, we can substitute r2r^2r2 with 111 to get x2+y2=1x^2 + y^2 = 1x2+y2=1. If we substitute x=cos⁡θx = \cos \thetax=cosθ and y=sin⁡θy = \sin \thetay=sinθ, we get the Pythagorean Identity which states that cos⁡2θ+sin⁡2θ=1\cos^2 \theta + \sin^2 \theta = 1cos2θ+sin2θ=1.

This function allows us to find the cosine of an angle if we know the sine of the angle and vice versa. However, due to the power of 222 in the Pythagorean Identity, we get both a positive and negative version of a solution even though we can only have one solution. So, we decide on the correct sign based on the quadrant in which the angle lies. For example, if the solution is in the first quadrant, then the sine and cosine functions are both positive because both the xxx and yyy coordinates are positive in the first quadrant. On the other hand, if the solution is in the fourth quadrant, then the sine function is negative because the yyy coordinate is negative and the cosine function is positive because the xxx coordinate is positive.

Example‾\underline{Example}Example​

If cos⁡(θ)=2425\cos(\theta) = \frac{24}{25}cos(θ)=2524​ and θ\thetaθ is in the fourth quadrant, find sin⁡(θ)\sin(\theta)sin(θ).

Using the Pythagorean Identity...

cos⁡2θ+sin⁡2θ=1(2425)2+sin⁡2θ=11−(2425)2=sin⁡2θ1−(576625)=sin⁡2θ49625=sin⁡2θ±725=sin⁡θ\begin{array}{lllll} \cos^2 \theta + \sin^2 \theta &=& 1 \\[1em] \left(\frac{24}{25}\right)^2 + \sin^2 \theta &=& 1 \\[1em] 1 - \left(\frac{24}{25}\right)^2 &=& \sin^2 \theta \\[1em] 1 - \left(\frac{576}{625}\right) &=& \sin^2 \theta \\[1em] \frac{49}{625} &=& \sin^2 \theta \\[1em] \pm \frac{7}{25} &=& \sin \theta \\[1em] \end{array}cos2θ+sin2θ(2524​)2+sin2θ1−(2524​)21−(625576​)62549​±257​​======​11sin2θsin2θsin2θsinθ​

Since θ\thetaθ is in the fourth quadrant, the sine function is negative. Therefore, sin⁡θ=−725\sin \theta = -\frac{7}{25}sinθ=−257​.

note

In trigonometry, writing a trigonometric function as fx(θ)f^x(\theta)fx(θ) is equivalent to writing (f(θ))x(f(\theta))^x(f(θ))x. This notation is a shorthand way of writing the trigonometric function raised to a power. For example, cos⁡2θ\cos^2 \thetacos2θ is equivalent to (cos⁡θ)2(\cos \theta)^2(cosθ)2.

Special Angles​

Using our definitions and the Pythagorean identity, we are able to find the exact values of the sine and cosine functions for the special angles. These angles include 0°0\degree0°, 30°30\degree30°, 45°45\degree45°, 60°60\degree60°, and 90°90\degree90° which are equivalent to 000, π6\frac{\pi}{6}6π​, π4\frac{\pi}{4}4π​, π3\frac{\pi}{3}3π​, and π2\frac{\pi}{2}2π​ radians respectively.

Proof: 45° Angle‾\underline{\text{Proof: } 45\degree \text{ Angle}}Proof: 45° Angle​

First, lets find the xxx and yyy coordinates of 45°45\degree45° or π4\frac{\pi}{4}4π​. A 45°45°90°45\degree 45\degree 90\degree45°45°90° triangle is an isosceles triangle, so the two legs are equal in length. This means for that in a unit circle, x=yx = yx=y.

We can use the pythagorean theorem to find the value of xxx and yyy for 45°45\degree45°...

x2+y2=1x2+x2=12x2=1x2=12x=12x=22\begin{array}{lllll} x^2 + y^2 &=& 1 \\[0.5em] x^2 + x^2 &=& 1 \\[0.5em] 2x^2 &=& 1 \\[0.5em] x^2 &=& \frac{1}{2} \\[0.5em] x &=& \frac{1}{\sqrt{2}} \\[0.5em] x &=& \frac{\sqrt{2}}{2} \\[0.5em] \end{array}x2+y2x2+x22x2x2xx​======​11121​2​1​22​​​

Due to the fact that x=yx = yx=y, we can say that y=22y = \frac{\sqrt{2}}{2}y=22​​. Therefore, cos⁡π4=22\cos \frac{\pi}{4} = \frac{\sqrt{2}}{2}cos4π​=22​​ and sin⁡π4=22\sin \frac{\pi}{4} = \frac{\sqrt{2}}{2}sin4π​=22​​. This also makes the coordinate of the point (22,22)(\frac{\sqrt{2}}{2}, \frac{\sqrt{2}}{2})(22​​,22​​).

We can also find the values of sine and cosine for 30°30\degree30° using some clever geometric manipulation...

Proof: 30° Angle‾\underline{\text{Proof: } 30\degree \text{ Angle}}Proof: 30° Angle​

To find the values of sin⁡\sinsin and cos⁡\coscos for 30°30\degree30° or π6\frac{\pi}{6}6π​, we can draw a triangle inside the unit circle with one side at an angle of 30°30\degree30° and another at an angle of −30°-30\degree−30°. The two right triangles should result in a triangle that can be combined into one large equailateral triangle where each angle is 60°60\degree60°.

Due to the fact that all the angles are equal, the sides are also equal. We know that one side of the triangle is rrr and another side is 2y2y2y. This means that r=2yr = 2yr=2y and y=r2y = \frac{r}{2}y=2r​. We also know r=1r = 1r=1 because the radius of the unit circle is 111. Therefore, y=12y = \frac{1}{2}y=21​ and so sin⁡(π6)=12\sin(\frac{\pi}{6}) = \frac{1}{2}sin(6π​)=21​.

Now that we know the value of sin(π6)sin(\frac{\pi}{6})sin(6π​), we can use the Pythagorean Identity to find the value of cos⁡(π6)\cos(\frac{\pi}{6})cos(6π​)...

cos⁡2θ+sin⁡2θ=1cos⁡2(π6)+sin⁡2(π6)=1cos⁡2(π6)+(12)2=1cos⁡2(π6)+14=1cos⁡2(π6)=34cos⁡(π6)=32\begin{array}{lllll} \cos^2 \theta + \sin^2 \theta &=& 1 \\[0.5em] \cos^2 (\frac{\pi}{6}) + \sin^2 (\frac{\pi}{6}) &=& 1 \\[0.5em] \cos^2 (\frac{\pi}{6}) + \left(\frac{1}{2}\right)^2 &=& 1 \\[0.5em] \cos^2 (\frac{\pi}{6}) + \frac{1}{4} &=& 1 \\[0.5em] \cos^2 (\frac{\pi}{6}) &=& \frac{3}{4} \\[0.5em] \cos (\frac{\pi}{6}) &=& \frac{\sqrt{3}}{2} \\[0.5em] \end{array}cos2θ+sin2θcos2(6π​)+sin2(6π​)cos2(6π​)+(21​)2cos2(6π​)+41​cos2(6π​)cos(6π​)​======​111143​23​​​

Therefore, cos⁡(π6)=32\cos(\frac{\pi}{6}) = \frac{\sqrt{3}}{2}cos(6π​)=23​​ and sin⁡(π6)=12\sin(\frac{\pi}{6}) = \frac{1}{2}sin(6π​)=21​. This also makes the coordinate of the point (32,12)(\frac{\sqrt{3}}{2}, \frac{1}{2})(23​​,21​).

For the 60°60\degree60° angle, we use the same 30°−60°−90°30\degree-60\degree-90\degree30°−60°−90° triangle in a similar way to find the values of sine and cosine.

Proof: 60° Angle‾\underline{\text{Proof: } 60\degree \text{ Angle}}Proof: 60° Angle​

To find the values of sin⁡\sinsin and cos⁡\coscos for 60°60\degree60° or π3\frac{\pi}{3}3π​, we can draw a right triangle BADBADBAD inside the unit circle with AAA at the origin with an angle of 60°60\degree60° and BBB at the point (x,y)(x, y)(x,y) with an angle of 30°30\degree30°. The hypotenuse of the triangle is the radius of the unit circle which is 111. The side opposite to the angle 60°60\degree60° is yyy and the side adjacent to the angle 60°60\degree60° is xxx.

We can draw another triangle BCDBCDBCD which is a reflection of triangle BADBADBAD over the line segment BDBDBD. When we combine the two triangles, we get an equailateral triangle ABCABCABC where each angle is 60°60\degree60°.

Since all the angles are equal, all the sides are equal. We know that one side is r=1r = 1r=1 and another side is 2x2x2x. This means that 1=2x1 = 2x1=2x and x=12x = \frac{1}{2}x=21​.

We can use the Pythagorean Theorem to find the value of yyy...

x2+y2=1(12)2+y2=114+y2=1y2=34y=32\begin{array}{lllll} x^2 + y^2 &=& 1 \\[0.5em] \left(\frac{1}{2}\right)^2 + y^2 &=& 1 \\[0.5em] \frac{1}{4} + y^2 &=& 1 \\[0.5em] y^2 &=& \frac{3}{4} \\[0.5em] y &=& \frac{\sqrt{3}}{2} \\[0.5em] \end{array}x2+y2(21​)2+y241​+y2y2y​=====​11143​23​​​

We ignore the negative value of yyy because the angle 60°60\degree60° is in the first quadrant. Therefore, cos⁡(π3)=12\cos(\frac{\pi}{3}) = \frac{1}{2}cos(3π​)=21​ and sin⁡(π3)=32\sin(\frac{\pi}{3}) = \frac{\sqrt{3}}{2}sin(3π​)=23​​. This also makes the coordinate of the point (12,32)(\frac{1}{2}, \frac{\sqrt{3}}{2})(21​,23​​).

Finally, the angles 0°0\degree0° and 90°90\degree90° lie on the xxx and yyy axes respectively. This means that the coordinates are (1,0)(1, 0)(1,0) and (0,1)(0, 1)(0,1) respectively. Therefore, cos⁡0°=1\cos 0\degree = 1cos0°=1, sin⁡0°=0\sin 0\degree = 0sin0°=0, cos⁡90°=0\cos 90\degree = 0cos90°=0, and sin⁡90°=1\sin 90\degree = 1sin90°=1. Reference Angles​

So far we have found the sine and cosine values for angles in the first quadrant. However, we can use the unit circle to find the sine and cosine values for angles in the other quadrants through the use of reference angles. A reference angle is the acute angle formed by the terminal side of the angle and the xxx-axis.

We can use the following formulas to find the reference angle of an angle θ\thetaθ...

Using Reference Angles​

The interesting thing about reference angles is that angles have consines and sines with the same absolute value as their reference angles. The sign of the specific cosine and sine function depends on the quadrant in which the angle lies. For example, if the angle is in the second quadrant, then the cosine function is negative because the xxx coordinate is negative in the second quadrant.

Example‾\underline{Example}Example​

Use the reference angle of −π6-\frac{\pi}{6}−6π​ to find cos⁡(−π6)\cos(-\frac{\pi}{6})cos(−6π​) and sin⁡(−π6)\sin(-\frac{\pi}{6})sin(−6π​).

To find the reference angle of −π6-\frac{\pi}{6}−6π​, lets make it positive by adding 2π2\pi2π to it. This gives us 11π6\frac{11\pi}{6}611π​.

Since 11π6\frac{11\pi}{6}611π​ is in the fourth quadrant, we can use θ′=2π−θ=2π−11π6=π6\theta' = 2\pi - \theta = 2\pi - \frac{11\pi}{6} = \frac{\pi}{6}θ′=2π−θ=2π−611π​=6π​ as the reference angle.

We know that cos⁡(π6)=32\cos(\frac{\pi}{6}) = \frac{\sqrt{3}}{2}cos(6π​)=23​​ and sin⁡(π6)=12\sin(\frac{\pi}{6}) = \frac{1}{2}sin(6π​)=21​. Since the angle is in the fourth quadrant, the cosine function is positive and the sine function is negative. Therefore, cos⁡(−π6)=32\cos(-\frac{\pi}{6}) = \frac{\sqrt{3}}{2}cos(−6π​)=23​​ and sin⁡(−π6)=−12\sin(-\frac{\pi}{6}) = -\frac{1}{2}sin(−6π​)=−21​.

Unit Circle​

Using the concept of reference angles and special angles, we can complete the unit circle by filling each quadrant with the sine and cosine values of the special angles.
