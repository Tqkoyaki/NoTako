Real Analysis

There are two types of numbers: discrete in which the set of numbers are finite and continuous where the set of numbers are infinite. There are various continuous sets we use often in math. These sets include natural numbers (N\mathbb{N}N), integers (Z\mathbb{Z}Z), rational numbers (Q\mathbb{Q}Q), and real numbers (R\mathbb{R}R). The focus of real analysis revolves around the set of real numbers.

Types of Numbers​

Not all infinite sets are created equal because N⊂Z⊂Q⊂R\mathbb{N} \subset \mathbb{Z} \subset \mathbb{Q} \subset \mathbb{R}N⊂Z⊂Q⊂R meaning R\mathbb{R}R encompasses more numbers than Q\mathbb{Q}Q and so on.

We can use these subsets to define a bigger set of numbers. For example, integers can be defined using natural numbers, rational numbers can be defined using integers, and so on.

Rational Numbers​

With integers being a subset of rational numbers, we are able to define rational numbers and their properties using integers. A rational number is any number that can be expressed as p/qp / qp/q where p,q∈Zp, q \in \mathbb{Z}p,q∈Z.

Rational numbers are considered dense because if we take any two numbers on the rational line, we can always find a new number on the rational line which falls in between the first two numbers we picked on the line. We can see this through the following proof:

Theorem: If r,s∈Qr, s \in \mathbb{Q}r,s∈Q and r<sr < sr<s, then there exists a rational ttt such that r<t<sr < t < sr<t<s.

Proof: Let t=12(r+s)t = \frac{1}{2} (r + s)t=21​(r+s). Clearly r<t<sr < t < sr<t<s.

But, is t∈Qt \in \mathbb{Q}t∈Q?

Let r=mn,s=pqr = \frac{m}{n}, s = \frac{p}{q}r=nm​,s=qp​, where m,n,p,q∈Zm, n, p, q \in \mathbb{Z}m,n,p,q∈Z.

Then t=12(mn+pq)=mq+np2nqt = \frac{1}{2} (\frac{m}{n} + \frac{p}{q}) = \frac{mq + np}{2nq}t=21​(nm​+qp​)=2nqmq+np​.

So mq+npmq + npmq+np, 2np∈Z2np \in \mathbb{Z}2np∈Z which means t∈Qt \in \mathbb{Q}t∈Q. Q.E.D.

Real Numbers​

As we have proven, rational numbers are dense. However, this does not prevent holes in the rational line. The line is dense but also full of holes like 2\sqrt{2}2​.

Let A=x∈Q∣x≤0∨x2<2A = {x \in \mathbb{Q} \mid x \leq 0 \lor x^2 < 2}A=x∈Q∣x≤0∨x2<2, and B=x∈Q∣x>0∧x2≥2B = {x \in \mathbb{Q} \mid x > 0 \land x^2 \geq 2}B=x∈Q∣x>0∧x2≥2.

Clearly, A∪B=QA \cup B = \mathbb{Q}A∪B=Q.

But, AAA has no greatest member and B has no smallest member. Hence, the rationals are inadequate to solve mathematics.

In Q\mathbb{Q}Q, we cannot solve the equation x2−2=0x^2 - 2 = 0x2−2=0.

These holes are why we have real numbers. Real numbers include all rational numbers and the holes in the rational line as well. This make this number set perfect for many mathematical and scientific uses.

Completeness Property​

The idea of real numbers filling the holes in the rational line is expressed through the completeness property. Before we can look at this property, we must look at the definition of a least upper bound. Given a set of A∈RA \in \mathbb{R}A∈R, a number bbb such that (∀a∈A)[a≤b](\forall a \in A)[a \leq b](∀a∈A)[a≤b] is said to be the upper bound of AAA. We say bbb is a least upper bound of AAA if, in addition, for any upper bound ccc of AAA, we have b≤cb \leq cb≤c.

The completeness property of the real number system says that every nonempty set of reals that has an upper bound, has a least upper bound in R\mathbb{R}R. Using this property, we can show that the rational line is not complete.
