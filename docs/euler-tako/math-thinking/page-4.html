Proofs

A proof is a logically sound argument that establishes the truth a given statement. Even if there is an intuition that some mathematical statement is true, we cannot be sure until we have proven it or read a proof that establishes the truth of that statement.

A good proof is one that:

  1. Has a logical structure that a reader could follow.
  2. Have thorough explanations that readers can understand.
  3. Be able to counter, explicitly or implicitly, all the counterarguments that readers may put forward.

Proof by Contradiction​

Proofs can be abstract but there are methods that have been commonly used to prove statements. One such method is proof by condradiction often used to prove something that doesn't exist where we show that the negation of the original statement is false and a true statement cannot lead to a false conclusion allowing us to disprove the negation. If the negation is false, then the original statement is true.

Theorem: 2\sqrt 22​ is irrational.

Proof: Assume on the contray, that 2\sqrt 22​ were rational.

Then there are natural numbers ppp, qqq, with no common factors, such that

2=p/q\sqrt 2 = p / q2​=p/q

Squaring: 2=p2/q22 = p^2 / q^22=p2/q2

Rearranging: 2q2=p22q^2 = p^22q2=p2

So p2p^2p2 is even. Hence ppp is even. So, p=2rp = 2rp=2r, for some rrr.

Substituting for ppp: 2q2=(2r)2=4r22q^2 = (2r)^2 = 4r^22q2=(2r)2=4r2.

Cancelling the 222: q2=2r2q^2 = 2r^2q2=2r2.

So q2q^2q2 is even. So qqq is even. So ppp, qqq are both even but this is impossible, since ppp, qqq has no common factors.

Hence the original assumption that 2\sqrt 22​ were rational must be false. Hence 2\sqrt 22​ must be irrational. Q.E.D.

note

In the proof above, the statement is a theorem because it has been proven and is significant.

Common statements we will encounter include...

  1. Theorems which are statements that have been proven or can be proven.
  2. Lemmas which are theorems that is not as significant.
  3. Conjectures are statements that has not been proven.

The general structure for a proof by contradiction is:

  1. You want to prove statement ϕ\phiϕ.
  2. You start by assuming ¬ϕ\neg\phi¬ϕ.
  3. You reason until you reach a conclusion that is false. This is often done by deducing both ψ\psiψ and ¬ψ\neg\psi¬ψ for some ψ\psiψ. For example, "ppp, qqq have no common factors" and "ppp, qqq are both even".
  4. A true assumption cannot lead to false conclusion.
  5. Hence the assumption ¬ϕ\neg\phi¬ϕ must be false.
  6. In other words, ϕ\phiϕ must be true.

Proving Conditionals​

When we are proving a conditional in the format ϕ⇒ψ\phi \Rightarrow \psiϕ⇒ψ, we know that the statement is true if ϕ\phiϕ is false. So, we can assume ϕ\phiϕ is true. To prove the statement, we assume ϕ\phiϕ and deduce ψ\psiψ. For example...

Claim: [ xxx, yyy are rational ] ⇒\Rightarrow⇒ [ x+yx + yx+y is rational ]

Proof: Assume xxx, yyy are rational.

Then there are integers ppp, qqq, mmm, nnn such that x=p/mx = p/mx=p/m, and y=q/ny = q/ny=q/n.

Then x+y=pm+qn=pn+qmmnx + y = \frac{p}{m} + \frac{q}{n} = \frac{pn + qm}{mn}x+y=mp​+nq​=mnpn+qm​.

Hence x+yx + yx+y is rational.

Quantifiers​

Sometimes when we are proving conditionals that involve quantifiers, it is best handled by proving the contrapositive. The contrapositive of ϕ⇒ψ\phi \Rightarrow \psiϕ⇒ψ is ¬ψ⇒¬ϕ\neg\psi \Rightarrow \neg\phi¬ψ⇒¬ϕ and these two statements are equivalent. An example of this kind of proof is...

Claim: (sinθ≠0)⇒(∀n∈N)(θ≠nπ)(sin \theta \neq 0) \Rightarrow (\forall n \in \mathbb{N}) (\theta \neq n \pi)(sinθ=0)⇒(∀n∈N)(θ=nπ)

Proof: The statement is equivalent to ¬(∀n∈N)(θ≠nπ)⇒¬(sin⁡θ≠0)\neg (\forall n \in \mathbb{N})(\theta \neq n \pi) \Rightarrow \neg (\sin \theta \neq 0)¬(∀n∈N)(θ=nπ)⇒¬(sinθ=0).

In positive form: (∃n∈N)(θ=nπ)⇒(sin⁡θ=0)(\exists n \in \mathbb{N})(\theta = n \pi) \Rightarrow (\sin \theta = 0)(∃n∈N)(θ=nπ)⇒(sinθ=0).

We know this is this! This proves the desired result.

Biconditionals​

In order to prove biconditionals in the format ϕ⇔ψ\phi \Leftrightarrow \psiϕ⇔ψ, we construct two proofs, one of ϕ⇒ψ\phi \Rightarrow \psiϕ⇒ψ and the other of ψ⇒ϕ\psi \Rightarrow \phiψ⇒ϕ. We have already looked at how to prove conditionals so we can apply the same principles to these proofs and the biconditional will be true if both proofs result in true statements. Finally, one thing to note is that occassionally it is easier to prove the two conditionals ϕ⇒ψ\phi \Rightarrow \psiϕ⇒ψ and ¬ϕ⇒¬ψ\neg\phi \Rightarrow \neg\psi¬ϕ⇒¬ψ. This works because ψ⇒ϕ\psi \Rightarrow \phiψ⇒ϕ is equivalent to ¬ϕ⇒¬ψ\neg\phi \Rightarrow \neg\psi¬ϕ⇒¬ψ.

Proof by Cases​

Proof by cases is a brute force method we use for statements that we can split into a finite number of cases and we test every case. An example of a proof using this method is...

Theorem: There are irrationals rrr, sss such that rsr^srs is rational.

Proof: We consider two cases,

Case 1: If 22\sqrt{2}^{\sqrt{2}}2​2​ is rational, r=s=2r = s = \sqrt{2}r=s=2​.

Case 2: If 22\sqrt{2}^{\sqrt{2}}2​2​ is irrational, r=22r = \sqrt{2}^{\sqrt{2}}r=2​2​ and s=2s = \sqrt{2}s=2​.

Then rs=(22)2=(2)2×2=(2)2=2r^s = (\sqrt{2}^{\sqrt{2}})^{\sqrt{2}} = (\sqrt{2})^{\sqrt{2} \times \sqrt{2}} = (\sqrt{2})^2 = 2rs=(2​2​)2​=(2​)2​×2​=(2​)2=2.

Proof by Induction​

We often use a powerful method called proof by induction to prove statements of the form ∀nA(n)\forall n A(n)∀nA(n) where n ranges over the natural numbers.

The principle of mathematical induction is that to prove ∀nA(n)\forall n A(n)∀nA(n), we need to establish the following two statements:

  1. A(1)A(1)A(1) which is th initial case / initial step.
  2. ∀n[A(n)⇒A(n+1)]\forall n [A(n) \Rightarrow A(n + 1)]∀n[A(n)⇒A(n+1)] which is the induction step.

Intuitively, this gives ∀nA(n)\forall n A(n)∀nA(n) as follows: By step 1, A(1)A(1)A(1). By step 2, A(1)⇒A(2)A(1) \Rightarrow A(2)A(1)⇒A(2). So from A(1)A(1)A(1) we can conclude A(2)A(2)A(2). By A(2)A(2)A(2) and the induction step, we can conclude A(3)A(3)A(3) and etc. Let's look at an example...

Theorem: For any nnn, 1+2+3+...+n=12n(n+1)1 + 2 + 3 + ... + n = \frac{1}{2} n (n + 1)1+2+3+...+n=21​n(n+1).

Proof: By mathematical induction.

For n=1n = 1n=1, the identity reduces to 1=12(1)(1+1)=12(1)(2)1 = \frac{1}{2} (1)(1 + 1) = \frac{1}{2} (1) (2)1=21​(1)(1+1)=21​(1)(2), which is true, since both sides equal 111.

Assume the identity holds for nnn, i.e. 1+2+...+n=12n(n+1)1 + 2 + ... + n = \frac{1}{2} n (n + 1)1+2+...+n=21​n(n+1). We want to deduce: 1+2+...+(n+1)=12(n+1)((n+1)+1)1 + 2 + ... + (n + 1) = \frac{1}{2}(n + 1)((n + 1) + 1)1+2+...+(n+1)=21​(n+1)((n+1)+1).

Add (n+1)(n + 1)(n+1) to both sides of our identity. We get 1+2+...+n+n+1=12n(n+1)+(n+1)1 + 2 + ... + n + n + 1 = \frac{1}{2} n (n + 1) + (n + 1)1+2+...+n+n+1=21​n(n+1)+(n+1)

12[n(n+1)+2(n+1)]=12[n2+n+2n+2]=12[n2+3n+2]=12[(n+1)(n+2)]=12(n+1)[(n+1)+1]\frac{1}{2}[n(n+1)+2(n+1)] = \frac{1}{2}[n^2 + n + 2n + 2] = \frac{1}{2}[n^2 + 3n + 2] = \frac{1}{2}[(n + 1)(n + 2)] = \frac{1}{2}(n + 1)[(n + 1) + 1]21​[n(n+1)+2(n+1)]=21​[n2+n+2n+2]=21​[n2+3n+2]=21​[(n+1)(n+2)]=21​(n+1)[(n+1)+1].

Which is the identity with n+1n + 1n+1 in place of nnn. Hence by the principle of mathematical induction, the identity holds for all nnn.

Mathematical induction is a powerful tool when it comes to proving universal statements. Usually, existence statements are easier to prove as you need to only find one case where the statement holds true. Universal statements on the other hand require us to prove all possible cases and so having tools like mathematical induction is very useful.
