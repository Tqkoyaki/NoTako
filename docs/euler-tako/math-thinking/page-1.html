Introduction

Mathematics is much more than just using procedures to solve problems. Modern mathematics has required deeper and deeper levels of understanding which is only achievable through mathematical thinking.

This subject can get extremely abstract past what our imagination can believe. One example of this is the Banach-Tarski Paradox which states that in theory you can take a sphere and cut it up and reassemble it in a way that you can get two identical spheres that are the same size as the original.

Our imagination will not believe such a notation but being able to understand and prove these abstract concepts, we need a precise system of language that we can follow. One that leaves no room for ambiguity and has the same meaning no matter the context. These are things that the Modern English Language fails to achieve.

Mathematical Statements​

As previously stated, math needs to be very precise and this is possible in math because language is used in a highly restricted way. In fact, almost all mathematical statements can be created through the variations of these four sentences...

  1. Object AAA has property PPP.
    • Ex. 333 is a prime number.
  2. Every object of type TTT has property PPP.
    • Ex. Every polynomial equation has a complex root.
  3. There is an object of type TTT having property.
    • Ex. There is a prime number between 202020 and 252525.
  4. If statement AAA, then statement BBB.
    • Ex. If PPP is a prime of the form 4n+14n + 14n+1 then PPP is a sum of two squares.

Almost all key statements of mathmatics, the axioms, conjectures, hypothesis, and theorems are positive or negative versions of one of these four linquistic forms. To build on this form, math also uses keywords which include: and, or, not, implies, for all, there exists.

Conjunction​

Conjuction is equivalent to andandand in English which enables us to combine two statements which we call the conjuncts. If both the conjuncts are true, then the conjunction is true. If either or both of the conjuncts are false, then the conjunction is false.

In mathematics, we denote conjunction with the symbol ∧\land∧. Let's say our two conjuncts are ψ\psiψ and ϕ\phiϕ then we denote the conjunction with ψ∧ϕ\psi \land \phiψ∧ϕ.

Disjunction​

Disjunction is equivalent to ororor in English where the combined statements are individually called the disjuncts. There are two kinds of ororor that we must consider. Both types of ororor are true if either of the disjuncts are true. Inclusive ororor is also true if both the statements are true. On the other hand, Exclusive ororor is false if both the statements are true.

In mathematics, it turns out, inclusive ororor tends to be more commonly used so when thats what ororor refers to on its own. The symbol we have given to ororor is ∨\lor∨. Let's say our two conjuncts are ψ\psiψ and ϕ\phiϕ then we can denote the disjunction with ψ∨ϕ\psi \lor \phiψ∨ϕ.

Negation​

The third type we will talk about is negation. This one is applied to a statement opposed to how we needed two statements for the others. Negation is equivalent to notnotnot in English and if the statement is true, negation makes it false. Vice versa if the statement is false, negation makes it true.

In mathematics, we denote negation with the symbol ¬\neg¬. If our statement is denoted by ψ\psiψ then the negation of that statement is ¬ψ\neg \psi¬ψ.

Proofs​

Now that we are able to create precise mathematical statements and are able to combine them, we can now look to determine if they are true or false. In mathematics, proofs are the primary means of determining the truth of any claim in mathematics. It is important to note that proofs can both be used to prove a statement true and a statement false. We can now look at two example proofs, one that proves a statement true and another that proves a statement false.

Euler's Proof​

The statement is: There are infinitely many prime numbers.

To prove this, we show that if we list the primes p1,p2,p3,...,p_1, p_2, p_3, ...,p1​,p2​,p3​,..., the list continues forever.

Suppose we have reached stage nnn: p1,p2,p3,...,pnp_1, p_2, p_3, ..., p_np1​,p2​,p3​,...,pn​. Can we find another prime to continue the list?

Look at the number N=(p1∗p2∗p3∗...∗pn)+1N = (p_1 * p_2 * p_3 * ... * p_n) + 1N=(p1​∗p2​∗p3​∗...∗pn​)+1.

If NNN is prime, we have found a prime bigger than pnp_npn​, and we can continue the list.

If NNN is not prime, it is divisible by a prime, say ppp. ppp cannot be any of p1,p2,...,pnp_1, p_2, ..., p_np1​,p2​,...,pn​ since dividing NNN by any of those leaves a remainder of 111.

So, p>pnp > p_np>pn​. This means we have found a prime bigger than pnp_npn​ again hence the list can always be continued.

This proves the statement is true.

Real Roots Proof​

The statement is: for every real number aaa, the equation x2+a=0x^2 + a = 0x2+a=0 has a real root.

We can prove this statement false by finding one case of aaa where the equation does not have a real root. If one case is incorrect, then the whole statement gets invalidated.

Let's take the case where aaa is 111. The equation x2+1=0x^2 + 1 = 0x2+1=0 does not have a real root because x=−1x = \sqrt{-1}x=−1​ is not a real number.

This proves the statement is false.
